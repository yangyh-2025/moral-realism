## v1.7 版本更新说明

### 1. 前端架构全面重构

**核心变更**：将前端架构从 Streamlit 迁移到 FastAPI + React 前后端分离架构，提供更强大的可扩展性和用户体验。

**架构升级**：
- **后端框架**：从 Streamlit 内置服务器升级为 FastAPI，提供专业的 API 服务
- **前端框架**：采用 React + TypeScript + Vite，支持模块化开发和组件复用
- **数据通信**：使用 RESTful API + WebSocket 实时通信替代 Streamlit Session State
- **部署方式**：支持 Docker 容器化部署，提供生产级部署方案

**技术优势**：
- 更好的性能：FastAPI 提供异步支持，处理高并发请求更高效
- 更强的可扩展性：前后端分离架构便于独立开发和维护
- 更丰富的 UI 组件库：React 生态系统提供大量成熟的 UI 组件
- 更好的开发体验：TypeScript 提供类型安全，Vite 提供快速开发和构建

### 2. API 接口标准化

**API 设计**：
- 遵循 RESTful 原则，提供标准化的接口设计
- 使用 Pydantic 模型进行请求和响应验证
- 完整的 OpenAPI 文档自动生成
- WebSocket 支持实时数据推送

**接口分类**：
- 仿真管理：启动、暂停、停止、重置仿真
- 智能体管理：配置、查询智能体信息
- 事件管理：事件配置、触发、查询
- 数据查询：历史数据、实时指标、互动记录
- 结果导出：支持多种格式数据导出

## v1.5 版本更新说明

### 1. 领导类型配置规则优化

根据国家实力层级调整领导类型配置要求：

**新配置规则**:
- **超级大国、大国**：必须配置领导类型（leader_type），因为它们具有全球领导能力
- **中等强国**：具有区域领导能力，但不需要配置全球领导类型
- **小国**：只有选择追随者的能力，不需要配置领导类型

**智能体初始化流程优化**:
1. 先设置所有智能体的基本参数（实力指标等）
2. 系统使用克莱因方程计算所有国家的综合国力
3. 使用正态分布方法动态划分实力层级
4. 根据国家实力层级提示用户需要配置哪些国家的领导人类型
5. 用户为超级大国和大国的国家选择领导人类型

**技术实现变更**:
- `BaseAgent.__init__` 不再要求提前传入leader_type
- 新增两步初始化流程：先创建智能体，计算实力层级后再配置领导类型
- `PowerTierClassifier` 在智能体创建后立即执行分类
- 前端界面增加"计算实力层级"按钮，先显示层级分布再配置领导类型

### 2. 记忆系统安全机制完善

确保每个国家只能读取其应该读取的内容，防止信息泄露：

**记忆访问控制机制**:
- **公开信息**：所有国家都可读取（公开声明、条约内容、公开互动等）
- **私下对话**：仅限参与对话的双方国家可读取，第三方不可见
- **内部决策理由**：仅做出决策的国家自身可读取（这是领导人的私人思考过程）
- **机密情报**：仅限指定国家可读取，根据情报权限控制

**记忆结构更新**:
- `AgentMemory` 新增 `private_conversations` 存储私下对话
- 新增 `classified_intelligence` 存储机密情报
- 每个记忆条目增加 `access_level` 和 `authorized_agents` 字段
- 新增 `get_memory_for_agent()` 方法，根据智能体权限过滤记忆内容

### 3. 行动理由记录与日志保密机制

所有国家智能体行动必须记录理由，但理由仅在日志中保存，不被其他国家知晓：

**日志结构更新**:
- 所有日志条目（DECISION/ACTION/SPEECH/JUDGMENT）都包含 `reasoning` 字段
- `reasoning` 字段标记为 `private`，仅在日志查询时可访问
- 向其他国家提供上下文时，自动过滤掉 `reasoning` 字段（这是领导人的私人思考过程）

**日志查询权限**:
- 系统管理员/研究者可查询完整日志（包含所有理由）
- 国家智能体查询其他国家信息时，看不到理由内容
- 导出功能支持选择是否包含理由（默认不包含）

## v1.4 版本更新说明

### 1. 克莱因方程实力测算系统

采用克莱因方程（Klein Equation）替代原有的硬实力/软实力加权平均方法，更科学地测算国家综合国力。

**克莱因方程公式**:
```
P = (C + E + M) × (S + W)
```

**克莱因方程结构说明**：
克莱因方程由美国学者克莱因提出，是将综合国力拆解为物质要素（硬实力）与精神要素（软实力）两大维度，二者相乘得出最终得分。综合国力总分1000分。

**一、物质要素（硬实力）**
物质要素由基本实体（C）、经济实力（E）、军事实力（M）三项构成，总分值为500分：

- **C - 基本实体（100分）**：人口和领土
  - 人口（50分）：反映国家人力资源基础
  - 领土面积（50分）：体现国家资源与空间禀赋

- **E - 经济实力（200分）**：
  - GNP（100分）：核心经济指标，量化国家经济总量
  - 能源、矿物、工业、食品、外贸各20分

- **M - 军事实力（200分）**：
  - 战略核力量（100分）
  - 常规力量（100分）

**二、精神要素（软实力）**
精神要素由战略目标（S）、国家意志（W）两项构成，总分值为1-2分：

- **S - 战略目标（0.5-1分）**：国家发展与安全的核心方向设定
- **W - 国家意志（0.5-1分）**：国民凝聚力、政策执行力等精神层面力量

该模型通过物质要素与精神要素的乘积关系，强调硬实力与软实力的协同作用，是国际关系领域综合国力评估的经典框架。

**三、实力层级动态分类方法（基于正态分布）**
为避免固定阈值在不同仿真场景中的不适用性，采用正态分布方法动态划分实力层级：

1. **计算所有智能体的克莱因P得分**：P = (C + E + M) × (S + W)
2. **计算样本统计量**：均值 μ 和标准差 σ
3. **标准化处理**：对每个智能体计算 z 分数：z = (P - μ) / σ
4. **根据z分数分类**（基于标准正态分布）：

| 层级 | z分数范围 | 理论比例 | 说明 |
|------|-----------|----------|------|
| 超级大国（superpowers） | z > 2.0 | ≈ 2.28% | 综合国力远超平均水平 |
| 大国（great powers） | 1.5 < z ≤ 2.0 | ≈ 4.41% | 综合国力显著高于平均水平 |
| 中等强国（middle powers） | 0.5 < z ≤ 1.5 | ≈ 24.17% | 综合国力高于平均水平 |
| 小国（small powers） | z ≤ 0.5 | ≈ 69.15% | 综合国力处于平均水平或以下 |

**技术实现变更**:
- `PowerMetrics` 类重构为克莱因方程五要素结构
- 新增 `calculate_material_power()` 和 `calculate_spiritual_power()` 方法
- `calculate_comprehensive_power()` 使用克莱因方程乘法公式
- 实力层级采用正态分布方法动态计算，不再使用固定阈值
- 新增 `PowerTierClassifier` 类实现基于正态分布的层级分类

**配置变更**:
- AgentConfig.power_metrics 结构更新为克莱因方程五要素
- 配置文件示例同步更新

## 一、技术方案总览

### 1.1 国际秩序类型完整定义（最终因变量分类）

基于道义现实主义理论框架，国际秩序类型作为仿真系统的最终因变量，需明确其核心治理模式、互动特征与判定标准，为体系层面仿真结果的量化分类提供依据。

| 秩序类型 | 英文标识 | 核心治理模式 | 互动核心特征 | 历史参考实例 |
|---------|---------|------------|------------|------------|
| 霸权秩序 | hegemonic | 由单一超级大国（霸权国）主导和维持，霸权国凭借超强综合实力制定并推行国际规则，提供安全、经济等国际公共产品，换取其他国家的服从与体系稳定 | 呈现「中心-外围」辐射状结构，其他国家与霸权国的关系为核心外交轴心，霸权国拥有规则最终解释权与仲裁权 | 19世纪不列颠治世、冷战后美国治世单极时期 |
| 均势秩序 | balance_of_power | 由多个实力相当的主要大国，通过结盟、制衡等手段确保无任何一国获得支配性优势，以此维持体系稳定 | 国家间互动以联盟政治和权力制衡为核心，外交合纵连横、动态调整，体系稳定依赖大国间的制衡默契 | 19世纪欧洲维也纳体系 |
| 规则/制度秩序 | rule_based | 由一套被广泛接受的国际规则、法律与制度主导和约束国家行为，权力因素在一定程度上被制度驯化 | 国家互动主要在多边机构和法律框架内进行，以主权平等、争端解决程序化、合作制度化为核心特征，大国需在规则内行事以维持合法性 | 二战后联合国宪章确立的国际体系、欧盟区域治理体系 |
| 无秩序型 | disorder | 国际规范全面失效，大国间无稳定制衡关系，暴力冲突成为国家间互动的主要方式，体系处于全面失序状态 | 无固定互动规则，丛林法则主导，国家间以零和博弈、生存竞争为核心，无稳定的联盟与合作关系 | 一战二战全面战争时期、帝国崩溃后的体系真空期 |
| 混合型秩序 | mixed | 以上三类稳定秩序中任意两类的组合模式，兼具两种秩序的核心特征 | 双轨/多轨互动模式，不同区域、不同议题领域遵循不同的治理规则，体系呈现碎片化特征 | 冷战时期两极格局下的全球秩序 |

**判定标准权重体系**：

| 秩序类型 | 主要判定指标 | 次要判定指标 | 判定逻辑 |
|---------|------------|------------|---------|
| 霸权秩序 | 实力集中度 (>60%) | 规范有效性 (>30%) | 高实力集中度 + 中等规范有效性 |
| 均势秩序 | 实力平衡 (30-60%) | 联盟数量 (≥2) | 中等实力集中度 + 多个有效联盟 + 中等规范有效性 |
| 规则/制度秩序 | 规范有效性 (>60%) | 制度化程度 (>60%) | 高规范有效性 + 高制度化程度 |
| 无秩序型 | 冲突水平 (>60%) | 规范有效性 (<30%) | 高冲突水平 + 低规范有效性 |
| 混合型秩序 | 区域多样性 | 议题碎片化 | 多个条件部分满足的复杂状态 |

**技术实现**：

```python
"""
LLM决策引擎 - 核心技术实现
"""
from abc import ABC, abstractmethod
from typing import Dict, List, Optional, Any
import httpx
import json

class LLMProvider(ABC):
    """LLM提供者抽象基类"""

    @abstractmethod
    async def generate(
        self,
        prompt: str,
        functions: List[Dict],
        temperature: float = 0.7,
        max_tokens: int = 2000
    ) -> Dict[str, Any]:
        pass

class OpenAIProvider(LLMProvider):
    """OpenAI兼容接口提供者"""

    def __init__(self, api_key: str, base_url: str, model: str):
        self.api_key = api_key
        self.base_url = base_url
        self.model = model
        self.client = httpx.AsyncClient(timeout=60.0)

    async def generate(
        self,
        prompt: str,
        functions: List[Dict],
        temperature: float = 0.7,
        max_tokens: int = 2000
    ) -> Dict[str, Any]:
        """调用LLM生成决策"""
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }

        payload = {
            "model": self.model,
            "messages": [{"role": "user", "content": prompt}],
            "functions": functions if functions else None,
            "temperature": temperature,
            "max_tokens": max_tokens
        }

        response = await self.client.post(
            f"{self.base_url}/chat/completions",
            headers=headers,
            json=payload
        )
        response.raise_for_status()
        return response.json()

class SiliconFlowProvider(LLMProvider):
    """SiliconFlow平台提供者（使用DeepSeek-V3.2模型）

    支持多API-key轮替调用，规避速率限制：
    1. 步骤2（各领导人决策并行进行）：多个API-key同时调用
    2. 其他步骤：多个API-key轮替调用
    """

    def __init__(
        self,
        api_key: Union[str, List[str]],
        base_url: str = "https://api.siliconflow.cn/v1",
        model: str = "deepseek-ai/DeepSeek-V3.2"
    ):
        # 支持单个API-key或多个API-key
        self.api_keys = [api_key] if isinstance(api_key, str) else api_key
        self.base_url = base_url
        self.model = model
        self.client = httpx.AsyncClient(timeout=60.0)
        # 轮替调用索引
        self._current_key_index = 0
        # 轮替调用锁
        self._key_lock = asyncio.Lock()

    def _get_next_api_key(self) -> str:
        """轮替获取下一个API-key"""
        key = self.api_keys[self._current_key_index]
        self._current_key_index = (self._current_key_index + 1) % len(self.api_keys)
        return key

    async def generate(
        self,
        prompt: str,
        functions: List[Dict],
        temperature: float = 0.7,
        max_tokens: int = 2000,
        use_rotation: bool = True  # 是否使用轮替调用
    ) -> Dict[str, Any]:
        """调用SiliconFlow API生成决策

        Args:
            prompt: 提示词
            functions: 可用函数列表
            temperature: 温度参数
            max_tokens: 最大token数
            use_rotation: 是否使用API-key轮替（步骤2设为False，其他步骤设为True）
        """
        # 选择API-key
        if use_rotation:
            async with self._key_lock:
                current_key = self._get_next_api_key()
        else:
            # 并行调用时，直接使用当前key索引对应的API-key
            async with self._key_lock:
                current_key = self.api_keys[self._current_key_index % len(self.api_keys)]
                self._current_key_index += 1

        headers = {
            "Authorization": f"Bearer {current_key}",
            "Content-Type": "application/json"
        }

        # 构建系统消息，要求LLM输出决策理由
        system_message = {
            "role": "system",
            "content": "你是国际关系模拟仿真系统中的智能决策助手。在做任何决策和判断时，必须提供充分的理由说明。输出格式：请先说明决策理由，再执行相应操作。"
        }

        payload = {
            "model": self.model,
            "messages": [system_message, {"role": "user", "content": prompt}],
            "functions": functions if functions else None,
            "temperature": temperature,
            "max_tokens": max_tokens
        }

        response = await self.client.post(
            f"{self.base_url}/chat/completions",
            headers=headers,
            json=payload
        )
        response.raise_for_status()
        return response.json()

class LocalLLMProvider(LLMProvider):
    """本地模型提供者（支持ollama等）"""

    def __init__(self, base_url: str, model: str):
        self.base_url = base_url
        self.model = model
        self.client = httpx.AsyncClient(timeout=120.0)

    async def generate(
        self,
        prompt: str,
        functions: List[Dict],
        temperature: float = 0.7,
        max_tokens: int = 2000
    ) -> Dict[str, Any]:
        """调用本地LLM生成决策"""
        headers = {"Content-Type": "application/json"}

        # 本地模型可能不支持function calling，需将functions转为prompt
        if functions:
            function_desc = "\n".join([
                f"- {f['name']}: {f['description']}"
                for f in functions
            ])
            prompt = f"{prompt}\n\n可用函数:\n{function_desc}\n\n请选择一个函数并返回函数名。"

        payload = {
            "model": self.model,
            "messages": [{"role": "user", "content": prompt}],
            "stream": False,
            "options": {
                "temperature": temperature,
                "num_predict": max_tokens
            }
        }

        response = await self.client.post(
            f"{self.base_url}/api/chat",
            headers=headers,
            json=payload
        )
        response.raise_for_status()
        return response.json()

class LLMEngine:
    """LLM决策引擎 - 统一接口"""

    def __init__(self, provider: LLMProvider):
        self.provider = provider
        self._cache = {}  # 简单缓存
        self._call_count = 0

    async def make_decision(
        self,
        agent_id: str,
        prompt: str,
        available_functions: List[Dict],
        prohibited_functions: List[str],
        use_rotation: bool = True  # 是否使用API-key轮替调用
    ) -> Dict[str, Any]:
        """
        生成智能体决策

        Args:
            agent_id: 智能体ID
            prompt: 完整的决策提示
            available_functions: 可用函数列表
            prohibited_functions: 禁止使用的函数列表
            use_rotation: 是否使用API-key轮替调用
                - False：步骤2并行决策，多个API-key同时调用
                - True：其他步骤，API-key轮替调用规避速率限制

        Returns:
            包含函数调用和参数的决策结果
        """
        # 过滤掉禁止使用的函数
        filtered_functions = [
            f for f in available_functions
            if f['name'] not in prohibited_functions
        ]

        # 生成决策（传递use_rotation参数）
        result = await self.provider.generate(
            prompt=prompt,
            functions=filtered_functions,
            temperature=0.7,
            max_tokens=2000,
            use_rotation=use_rotation
        )

        self._call_count += 1

        # 提取函数调用
        if 'function_call' in result.get('choices', [{}])[0].get('message', {}):
            function_call = result['choices'][0]['message']['function_call']
            return {
                'function': function_call['name'],
                'arguments': json.loads(function_call['arguments'])
            }

        # 如果没有函数调用，返回原始文本
        return {
            'function': None,
            'text': result.get('choices', [{}])[0].get('message', {}).get('content', '')
        }

    async def batch_make_decisions(
        self,
        requests: List[Dict[str, Any]],
        use_rotation: bool = True
    ) -> List[Dict[str, Any]]:
        """
        批量生成决策（并行处理）

        Args:
            requests: 决策请求列表，每个请求包含agent_id, prompt,
                     available_functions, prohibited_functions
            use_rotation: 是否使用API-key轮替调用

        Returns:
            决策结果列表
        """
        import asyncio

        tasks = [
            self.make_decision(
                agent_id=req['agent_id'],
                prompt=req['prompt'],
                available_functions=req['available_functions'],
                prohibited_functions=req['prohibited_functions'],
                use_rotation=use_rotation
            )
            for req in requests
        ]

        return await asyncio.gather(*tasks)
```

```python
            '%(asctime)s - %(name)s - %(levelname)s - [%(simulation_id)s] [Round:%(round)s] [Agent:%(agent_id)s] %(message)s'
        )

        file_handler = logging.FileHandler(log_file, encoding='utf-8')
        file_handler.setFormatter(formatter)

        # 控制台处理器
        console_handler = logging.StreamHandler()
        console_handler.setFormatter(formatter)

        # 配置根日志器
        self.logger = logging.getLogger("ABM_Simulation")
        self.logger.setLevel(logging.DEBUG)
        self.logger.addHandler(file_handler)
        self.logger.addHandler(console_handler)

        if agent_id:
            logs = [log for log in logs if log.agent_id == agent_id]

        # 转换为字典，支持理由过滤
        return [log.to_dict(include_reasoning=include_reasoning) for log in logs]

        self,
        simulation_id: str
    ) -> Dict[str, Any]:

        获取统计数据摘要

        Args:
            simulation_id: 仿真ID

        Returns:
            统计摘要

            return {}

        stats_by_category = {}
        for stat in self._statistics[simulation_id]:
            if stat.category not in stats_by_category:
                stats_by_category[stat.category] = []
            stats_by_category[stat.category].append(stat)

        return {
            "simulation_id": simulation_id,
            "categories": stats_by_category,
            "total_records": len(self._statistics[simulation_id])
        }

    def export_logs_to_file(
        self,

            output_path = str(self.log_dir / f"{simulation_id}_logs{suffix}.json")

            include_reasoning=include_reasoning

            "export_time": datetime.now().isoformat(),
            "total_logs": len(logs),
            "include_reasoning": include_reasoning,
            "logs": logs
        }

        with open(output_path, 'w', encoding='utf-8') as f:
            json.dump(export_data, f, ensure_ascii=False, indent=2)

        return output_path
```

#### 3.2.3 规则校验引擎 (core/validator.py)

```python
"""
规则校验引擎 - 验证所有行为符合道义现实主义规则
"""
from typing import Dict, List, Tuple, Optional
from dataclasses import dataclass
from enum import Enum

class LeaderType(str, Enum):
    """领导类型枚举"""
    WANGDAO = "wangdao"      # 王道型
    BAQUAN = "baquan"        # 霸权型
    QIANGQUAN = "qiangquan"  # 强权型
    HUNYONG = "hunyong"      # 昏庸型

@dataclass
class ValidationResult:
    """校验结果"""
    is_valid: bool
    error_message: str
    corrected_function: Optional[str] = None
    corrected_args: Optional[Dict] = None

class RuleValidator:
    """规则校验引擎"""

    # 王道型绝对禁止的函数
    WANGDAO_FORBIDDEN = {
        'use_military_force',
        'unilateral_sanctions',
        'unilateral_treaty_withdrawal',
        'double_standard_action',
        'sacrifice_ally_interest'
    }

    # 霸权型绝对禁止的函数
    BAQUAN_FORBIDDEN = {
        'full_equal_negotiation',
        'unconditionally_follow_all_norms',
        'sacrifice_core_interest_for_stability'
    }

    # 强权型绝对禁止的函数
    QIANGQUAN_FORBIDDEN = {
        'international_mediation',
        'provide_public_goods',
        'equal_multilateral_cooperation',
        'follow_unfavorable_norms'
    }

    # 昏庸型无禁止项

    @staticmethod
    def validate_decision(
        leader_type: LeaderType,
        function_name: str,
        function_args: Dict,
        objective_interest: str,
        current_power: float
    ) -> ValidationResult:
        """
        验证决策是否符合规则

        Args:
            leader_type: 领导类型
            function_name: 函数名
            function_args: 函数参数
            objective_interest: 客观战略利益
            current_power: 当前实力

        Returns:
            校验结果
        """
        # 1. 校验函数调用权限
        func_validation = RuleValidator._validate_function_permission(
            leader_type, function_name
        )
        if not func_validation.is_valid:
            return func_validation

        # 2. 校验实力变动约束
        if 'power_change' in function_args:
            power_change_validation = RuleValidator._validate_power_change(
                current_power,
                function_args['power_change']
            )
            if not power_change_validation.is_valid:
                return power_change_validation

        # 3. 校验战略匹配度（非昏庸型）
        if leader_type != LeaderType.HUNYONG:
            match_validation = RuleValidator._validate_strategic_match(
                function_name,
                function_args,
                objective_interest,
                leader_type
            )
            if not match_validation.is_valid:
                return match_validation

        return ValidationResult(True, "")

    @staticmethod
    def _validate_function_permission(
        leader_type: LeaderType,
        function_name: str
    ) -> ValidationResult:
        """校验函数调用权限"""
        forbidden_set = None

        if leader_type == LeaderType.WANGDAO:
            forbidden_set = RuleValidator.WANGDAO_FORBIDDEN
        elif leader_type == LeaderType.BAQUAN:
            forbidden_set = RuleValidator.BAQUAN_FORBIDDEN
        elif leader_type == LeaderType.QIANGQUAN:
            forbidden_set = RuleValidator.QIANGQUAN_FORBIDDEN
        elif leader_type == LeaderType.HUNYONG:
            # 昏庸型无禁止项
            return ValidationResult(True, "")

        if forbidden_set and function_name in forbidden_set:
            return ValidationResult(
                False,
                f"{leader_type}型领导禁止使用函数: {function_name}"
            )

        return ValidationResult(True, "")

    @staticmethod
    def _validate_power_change(
        current_power: float,
        power_change: float
    ) -> ValidationResult:
        """校验实力变动约束（5年不超过5%）"""
        # 实际实现需要跟踪历史实力变化
        max_allowed_change = 5.0  # 5年累计不超过5%

        if abs(power_change) > max_allowed_change:
            return ValidationResult(
                False,
                f"实力变动超出约束: {power_change}% > {max_allowed_change}%"
            )

        return ValidationResult(True, "")

    @staticmethod
    def _validate_strategic_match(
        function_name: str,
        function_args: Dict,
        objective_interest: str,
        leader_type: LeaderType
    ) -> ValidationResult:
        """校验战略匹配度"""
        # 简化实现：根据函数名和客观利益的匹配度计算
        # 实际需要更复杂的语义匹配

        # 非昏庸型决策需与客观利益有一定匹配度
        # 这里使用简化的关键词匹配
        match_score = RuleValidator._calculate_match_score(
            function_name, objective_interest
        )

        if match_score < 0.3:  # 匹配度低于30%
            return ValidationResult(
                False,
                f"决策与客观利益匹配度过低: {match_score:.2f}"
            )

        return ValidationResult(True, "")

    @staticmethod
    def _calculate_match_score(
        function_name: str,
        objective_interest: str
    ) -> float:
        """计算决策与客观利益的匹配度（简化实现）"""
        # 实际实现需要语义分析或关键词匹配
        keywords = {
            'maintain_global_leadership': ['dominance', 'leadership', 'global'],
            'expand_influence': ['influence', 'power', 'global'],
            'regional_dominance': ['regional', 'area', 'stability'],
            'security_protection': ['security', 'sovereignty', 'safety']
        }

        if function_name in keywords:
            for keyword in keywords[function_name]:
                if keyword.lower() in objective_interest.lower():
                    return 0.8

        return 0.5  # 默认中等匹配度
```

```python
from enum import Enum

class PowerTier(str, Enum):
    """实力层级枚举

    采用正态分布方法动态划分，基于z分数：
    - SUPERPOWER: 超级大国 (z > 2.0)，理论比例 ≈ 2.28%
    - GREAT_POWER: 大国 (1.5 < z ≤ 2.0)，理论比例 ≈ 4.41%
    - MIDDLE_POWER: 中等强国 (0.5 < z ≤ 1.5)，理论比例 ≈ 24.17%
    - SMALL_POWER: 小国 (z ≤ 0.5)，理论比例 ≈ 69.15%

    GREAT_POWER = "great_power"    # 大国
    MIDDLE_POWER = "middle_power"  # 中等强国
    SMALL_POWER = "small_power"    # 小国

@dataclass
class PowerMetrics:
    """实力指标 - 克莱因方程五要素模型

    克莱因方程公式：P = (C + E + M) × (S + W)

    物质要素（硬实力）：
    - C (基本实体): 0-100分（人口50分 + 领土50分）
    - E (经济实力): 0-200分（GNP 100分 + 其他5项各20分）
    - M (军事实力): 0-200分（战略核力量100分 + 常规力量100分）

    精神要素（软实力）：
    - S (战略目标): 0.5-1分
    - W (国家意志): 0.5-1分

    综合国力总分：物质要素最大500分 × 精神要素最大2分 = 1000分
    """
    # 物质要素子指标
    critical_mass: float          # C - 基本实体 0-100分
    economic_capability: float    # E - 经济实力 0-200分
    military_capability: float    # M - 军事实力 0-200分

    # 精神要素子指标
    strategic_purpose: float      # S - 战略目标 0.5-1分
    national_will: float          # W - 国家意志 0.5-1分

    def calculate_material_power(self) -> float:
        """计算物质要素实力 (C+E+M)"""
        return self.critical_mass + self.economic_capability + self.military_capability

    def calculate_spiritual_power(self) -> float:
        """计算精神要素实力 (S+W)"""
        return self.strategic_purpose + self.national_will

    def calculate_comprehensive_power(self) -> float:
        """计算综合实力指数 - 克莱因方程"""
        # P = (C + E + M) × (S + W)
        material_power = self.calculate_material_power()
        spiritual_power = self.calculate_spiritual_power()
        return material_power * spiritual_power

    def get_power_tier(self) -> PowerTier:
        """
        获取实力层级 - 已废弃，改用PowerTierClassifier动态分类

        注意：此方法不再使用，所有智能体的层级应在初始化时通过

        """

class PowerTierClassifier:
    """实力层级分类器 - 基于正态分布方法"""

    @staticmethod
    def classify_all(power_metrics_list: List[PowerMetrics]) -> List[PowerTier]:
        """
        对所有智能体的实力进行批量分类

        步骤：
        1. 计算所有智能体的克莱因P得分：P = (C + E + M) × (S + W)
        2. 计算样本均值 μ 和标准差 σ
        3. 对每个智能体标准化为 z 分数：z = (P - μ) / σ
        4. 根据z分数分类

        Args:
            power_metrics_list: 所有智能体的PowerMetrics列表

        Returns:
            对应的实力层级列表
        """
        import numpy as np

        # 步骤1: 计算所有P得分
        power_scores = [pm.calculate_comprehensive_power() for pm in power_metrics_list]

        if len(power_scores) == 0:
            return []

        # 步骤2: 计算均值和标准差
        mu = np.mean(power_scores)
        sigma = np.std(power_scores)

        # 避免除以零（如果所有智能体实力相同）
        if sigma < 1e-10:
            sigma = 1.0

        # 步骤3-4: 根据z分数分类
        tiers = []
        for score in power_scores:
            z = (score - mu) / sigma
            tier = PowerTierClassifier._classify_by_z_score(z)
            tiers.append(tier)

        return tiers

    @staticmethod
    def _classify_by_z_score(z: float) -> PowerTier:
        """
        根据z分数分类（基于标准正态分布）

        | 层级 | z分数范围 | 理论比例 |
        |------|-----------|----------|
        | 超级大国 | z > 2.0 | ≈ 2.28% |
        | 大国 | 1.5 < z ≤ 2.0 | ≈ 4.41% |
        | 中等强国 | 0.5 < z ≤ 1.5 | ≈ 24.17% |
        | 小国 | z ≤ 0.5 | ≈ 69.15% |

        Args:
            z: 标准化后的z分数

        Returns:
            实力层级
        """
        if z > 2.0:
            return PowerTier.SUPERPOWER
        elif z > 1.5:
            return PowerTier.GREAT_POWER
        elif z > 0.5:
            return PowerTier.MIDDLE_POWER
        else:
            return PowerTier.SMALL_POWER

class PowerSystem:
    """实力计算系统"""

    # 客观战略利益定义（对应方案2.2.3节）

    def validate_power_change(
        current_power: float,
        power_change: float,
        years_elapsed: int
    ) -> Tuple[bool, str]:
        """
        验证实力变动是否合规（5年不超过5%）

        Args:
            current_power: 当前实力
            power_change: 实力变动值
            years_elapsed: 经过年数

        Returns:
            (是否合规, 错误信息)

        if abs(power_change) > max_allowed_change:
            return (
                False,
                f"实力变动超出约束: {power_change:.2f}% > "
                f"允许的{max_allowed_change:.2f}% ({years_elapsed}年)"
            )

        return True, ""

        initial_power: float,
        current_power: float,
        rounds_elapsed: int

        Args:
            initial_power: 初始实力
            current_power: 当前实力
            rounds_elapsed: 经过的轮次

        Returns:
            稳定性评分 0-100
        """
        change_ratio = abs(current_power - initial_power) / max(initial_power, 1.0)
        expected_max_change = (rounds_elapsed / 20.0) * 0.05  # 假设20轮=5年

        if change_ratio <= expected_max_change:
            return 100.0
        else:
            excess_ratio = change_ratio - expected_max_change
            # 每超出1%扣除10分
            return max(0.0, 100.0 - excess_ratio * 1000.0)
```

```python
    PRIVATE = "private"          # 私有信息，仅自己可见（领导人决策理由）
    CLASSIFIED = "classified"    # 机密情报，仅限授权国家可见

@dataclass
class MemoryEntry:
    """记忆条目基类"""
    entry_id: str
    content: str
    timestamp: datetime = field(default_factory=datetime.now)
    access_level: AccessLevel = AccessLevel.PUBLIC
    authorized_agents: List[str] = field(default_factory=list)  # 可访问的智能体ID列表
    source_agent: Optional[str] = None  # 来源智能体ID

    def can_access(self, agent_id: str) -> bool:
        """检查指定智能体是否有权限访问此记忆"""
        if self.access_level == AccessLevel.PUBLIC:
            return True
        if self.access_level == AccessLevel.PRIVATE:
            return agent_id == self.source_agent
        if self.access_level == AccessLevel.RESTRICTED:
            return agent_id == self.source_agent or agent_id in self.authorized_agents
        if self.access_level == AccessLevel.CLASSIFIED:
            return agent_id in self.authorized_agents
        return False

@dataclass
class ConversationEntry(MemoryEntry):
    """对话记录"""
    conversation_type: str = "public"  # public 或 private
    participants: List[str] = field(default_factory=list)  # 参与对话的智能体ID
    is_private: bool = False

@dataclass
class AgentMemory:
    """国家记忆系统 - 带访问控制的完整记忆系统

    记忆安全规则:
    - 公开信息：所有国家都可读取
    - 私下对话：仅限参与对话的双方国家可读取
    - 内部决策理由：仅该国领导人自身可读取（私人思考过程）
    - 机密情报：仅限指定国家可读取
    """
    # 国家ID（用于权限验证）
    self_agent_id: str = ""

    # ========== 长期记忆 ==========
    # 公开国际承诺（所有国家可见）
    international_commitments: Dict[str, MemoryEntry] = field(default_factory=dict)
    # 签署的条约（所有国家可见）
    signed_treaties: List[MemoryEntry] = field(default_factory=list)
    # 历史行为记录（公开部分）
    historical_behavior: List[MemoryEntry] = field(default_factory=list)

    # ========== 短期记忆 ==========
    # 公开互动记录
    recent_interactions: List[MemoryEntry] = field(default_factory=list)
    # 私下对话记录（仅限参与方可见）
    private_conversations: List[ConversationEntry] = field(default_factory=list)
    # 决策记录（包含理由，但理由是私有的）
    recent_decisions: List[Dict] = field(default_factory=list)

    # ========== 机密情报 ==========
    # 机密情报存储（仅限授权访问）
    classified_intelligence: Dict[str, MemoryEntry] = field(default_factory=dict)

    def add_interaction(
        self,
        interaction: Dict,
        access_level: AccessLevel = AccessLevel.PUBLIC,
        authorized_agents: Optional[List[str]] = None
    ) -> None:
        """添加互动记录

        Args:
            interaction: 互动内容
            access_level: 访问级别
            authorized_agents: 可访问此记录的智能体ID列表（仅RESTRICTED和CLASSIFIED需要）
        """
        entry = MemoryEntry(
            entry_id=f"interaction_{len(self.recent_interactions)}",
            content=json.dumps(interaction, ensure_ascii=False),
            access_level=access_level,
            authorized_agents=authorized_agents or [],
            source_agent=self.self_agent_id
        )
        self.recent_interactions.append(entry)
        # 限制短期记忆大小
        if len(self.recent_interactions) > 20:
            self.recent_interactions = self.recent_interactions[-20:]

    def add_private_conversation(
        self,
        content: str,
        participant_ids: List[str],
        is_initiator: bool = True
    ) -> None:
        """添加私下对话记录（仅限参与双方可见）

        Args:
            content: 对话内容
            participant_ids: 参与对话的其他智能体ID
            is_initiator: 是否是发起方
        """
        all_participants = [self.self_agent_id] + participant_ids
        entry = ConversationEntry(
            entry_id=f"conv_{datetime.now().strftime('%Y%m%d_%H%M%S')}",
            content=content,
            access_level=AccessLevel.RESTRICTED,
            authorized_agents=all_participants,
            source_agent=self.self_agent_id,
            conversation_type="private",
            participants=all_participants,
            is_private=True
        )
        self.private_conversations.append(entry)
        if len(self.private_conversations) > 10:
            self.private_conversations = self.private_conversations[-10:]

    def add_decision(
        self,
        decision: Dict,
        reasoning: Optional[str] = None
    ) -> None:
        """添加决策记录

        Args:
            decision: 决策内容（公开）
            reasoning: 决策理由（私有，仅自己可见）
        """
        # 决策内容是公开的，但理由是私有的
        decision_record = {
            "decision": decision,
            "reasoning": reasoning,
            "timestamp": datetime.now(),
            "is_reasoning_private": True  # 标记理由为私有
        }
        self.recent_decisions.append(decision_record)
        if len(self.recent_decisions) > 10:
            self.recent_decisions = self.recent_decisions[-10:]

    def add_commitment(
        self,
        commitment_id: str,
        content: str,
        is_public: bool = True
    ) -> None:
        """添加国际承诺"""
        entry = MemoryEntry(
            entry_id=commitment_id,
            content=content,
            access_level=AccessLevel.PUBLIC if is_public else AccessLevel.RESTRICTED,
            source_agent=self.self_agent_id
        )
        self.international_commitments[commitment_id] = entry

    def add_classified_intelligence(
        self,
        intel_id: str,
        content: str,
        authorized_agents: List[str]
    ) -> None:
        """添加机密情报

        Args:
            intel_id: 情报ID
            content: 情报内容
            authorized_agents: 可访问此情报的智能体ID列表
        """
        entry = MemoryEntry(
            entry_id=intel_id,
            content=content,
            access_level=AccessLevel.CLASSIFIED,
            authorized_agents=authorized_agents,
            source_agent=self.self_agent_id
        )
        self.classified_intelligence[intel_id] = entry

    def remove_commitment(self, commitment_id: str) -> None:
        """移除国际承诺"""
        if commitment_id in self.international_commitments:
            del self.international_commitments[commitment_id]

    def get_memory_for_agent(self, querying_agent_id: str) -> Dict:
        """获取指定智能体可以访问的记忆内容

        这是记忆系统的安全核心：根据查询者的ID过滤记忆内容，
        确保每个智能体只能读取其应该读取的内容。

        Args:
            querying_agent_id: 查询记忆的智能体ID

        Returns:
            过滤后的记忆内容字典
        """
        result = {
            "international_commitments": {},
            "signed_treaties": [],
            "historical_behavior": [],
            "recent_interactions": [],
            "recent_decisions": []  # 注意：不包含理由，只包含决策内容
        }

        # 过滤国际承诺
        for cid, entry in self.international_commitments.items():
            if entry.can_access(querying_agent_id):
                result["international_commitments"][cid] = entry.content

        # 过滤条约
        for entry in self.signed_treaties:
            if entry.can_access(querying_agent_id):
                result["signed_treaties"].append(entry.content)

        # 过滤历史行为
        for entry in self.historical_behavior:
            if entry.can_access(querying_agent_id):
                result["historical_behavior"].append(entry.content)

        # 过滤近期互动
        for entry in self.recent_interactions:
            if entry.can_access(querying_agent_id):
                result["recent_interactions"].append(json.loads(entry.content))

        # 过滤决策记录（只返回决策内容，不返回理由）
        for decision_record in self.recent_decisions:
            # 只返回公开的决策内容，不返回理由
            result["recent_decisions"].append(decision_record["decision"])

        # 私下对话和机密情报不会出现在这里，
        # 只能通过专门的方法获取，且仅限授权方

        return result

    def get_private_conversation_with(
        self,
        other_agent_id: str,
        querying_agent_id: str
    ) -> List[ConversationEntry]:
        """获取与指定国家的私下对话（仅限参与方）"""
        if querying_agent_id not in [self.self_agent_id, other_agent_id]:
            # 非参与方无权查看
            return []

        return [
            entry for entry in self.private_conversations
            if other_agent_id in entry.participants
        ]

    def get_full_memory(self) -> Dict:
        """获取完整记忆（仅自己可用，用于LLM决策时提供完整上下文）

        注意：这个方法只能被记忆所属的智能体调用，
        返回的内容包含私有信息和理由，不应分享给其他智能体。
        """
        return {
            "international_commitments": {
                cid: entry.content
                for cid, entry in self.international_commitments.items()
            },
            "signed_treaties": [entry.content for entry in self.signed_treaties],
            "historical_behavior": [entry.content for entry in self.historical_behavior],
            "recent_interactions": [
                json.loads(entry.content)
                for entry in self.recent_interactions
            ],
            "private_conversations": self.private_conversations,
            "recent_decisions": self.recent_decisions,  # 包含理由
            "classified_intelligence": {
                iid: entry.content
                for iid, entry in self.classified_intelligence.items()
            }
        }

@dataclass
class AgentState:
    """智能体状态"""
    agent_id: str
    name: str
    agent_type: str
    region: str

    # 实力属性
    power_metrics: PowerMetrics
    power_tier: PowerTier

    # 领导属性 - 仅超级大国、大国需要配置
    # 中等强国只有区域领导能力，小国只有选择追随者的能力，不需要配置
    leader_type: Optional[LeaderType] = None
    core_preferences: Dict[str, float] = field(default_factory=dict)
    behavior_boundaries: List[str] = field(default_factory=list)

    # 记忆 - 带访问控制的记忆系统
    memory: AgentMemory = field(default_factory=lambda: AgentMemory())

    # 统计数据
    decision_count: int = 0
    function_call_history: Dict[str, int] = field(default_factory=dict)
    strategic_reputation: float = 100.0  # 战略信誉度

    # 客观战略利益
    objective_interest: str = ""

class BaseAgent(ABC):
    """智能体基类 - 支持两步初始化流程

    初始化流程（正确做法）:
    1. 先创建所有国家的基本实例，只设置客观参数（国家名称、区域、国家实力指标等）
    2. 系统计算所有国家的综合国力，动态划分国家实力层级
    3. 根据国家实力层级配置领导人类型（仅超级大国和大国需要）
    4. 完成最终初始化

    这样可以避免预先猜测哪些国家是大国，让系统计算后再做配置决策。
    """

    def __init__(
        self,
        agent_id: str,
        name: str,
        region: str,
        power_metrics: PowerMetrics
    ):
        """
        第一步初始化：创建国家智能体的基本实例

        只设置国家的客观参数，不依赖国家实力层级和领导人类型，
        因为这些需要在系统计算完所有国家实力指标后才能确定。

        Args:
            agent_id: 智能体唯一ID（代表国家）
            name: 国家名称
            region: 国家所属区域
            power_metrics: 国家的实力指标数据
        """
        # 保存原始配置，等待实力层级计算后初始化状态
        self._init_config = {
            "agent_id": agent_id,
            "name": name,
            "region": region,
            "power_metrics": power_metrics,
            "leader_type": None  # 初始设为None，等待配置
        }

        # 临时状态占位
        self.state = None
        self.power_tier = None
        self._is_initialized = False

    def set_leader_type(self, leader_type: Optional[LeaderType]) -> None:
        """
        设置领导人类型（在国家实力层级确定后调用）

        Args:
            leader_type: 领导人类型

        Raises:
            ValueError: 如果国家实力层级与领导人类型不匹配
        """
        if self.power_tier is None:
            raise ValueError("必须先完成国家实力层级计算，才能设置领导人类型")

        if self._is_initialized:
            raise ValueError("智能体已完成初始化，无法再设置领导人类型")

        if self.power_tier in [PowerTier.SUPERPOWER, PowerTier.GREAT_POWER]:
            if leader_type is None:
                raise ValueError(
                    f"{self._init_config['name']}({self.power_tier.value})是超级大国或大国，"
                    "必须配置领导类型"
                )
            self._init_config["leader_type"] = leader_type
        elif self.power_tier in [PowerTier.MIDDLE_POWER, PowerTier.SMALL_POWER]:
            if leader_type is not None:
                raise ValueError(
                    f"{self._init_config['name']}({self.power_tier.value})是中等强国或小国，"
                    "不需要配置领导类型"
                )
            self._init_config["leader_type"] = None

    def complete_initialization(self) -> None:
        """
        完成最终初始化（在设置领导类型后调用）

        必须按顺序调用：
        1. __init__()
        2. calculate_power_tier()
        3. set_leader_type() (如需要)
        4. complete_initialization()
        """
        if self.power_tier is None:
            raise ValueError("必须先调用 calculate_power_tier() 计算实力层级")

        if self._is_initialized:
            raise ValueError("智能体已完成初始化")

        init_data = self._init_config
        power_metrics = init_data["power_metrics"]
        leader_type = init_data["leader_type"]
        name = init_data["name"]

        # 验证领导类型与实力层级的匹配
        if self.power_tier in [PowerTier.SUPERPOWER, PowerTier.GREAT_POWER]:
            if leader_type is None:
                raise ValueError(
                    f"{name}({self.power_tier.value})必须配置leader_type（领导类型）"
                )
        elif self.power_tier in [PowerTier.MIDDLE_POWER, PowerTier.SMALL_POWER]:
            if leader_type is not None:
                raise ValueError(
                    f"{name}({self.power_tier.value})是中等强国或小国，不需要配置领导类型"
                )

        # 创建正式状态，包含带有访问控制的记忆系统
        self.state = AgentState(
            agent_id=init_data["agent_id"],
            name=name,
            agent_type=self.__class__.__name__,
            region=init_data["region"],
            power_metrics=power_metrics,
            power_tier=self.power_tier,
            leader_type=leader_type,
            core_preferences=self._get_core_preferences(leader_type) if leader_type else self._get_small_state_preferences(),
            behavior_boundaries=self._get_behavior_boundaries(leader_type) if leader_type else self._get_small_state_boundaries(),
            memory=AgentMemory(self_agent_id=init_data["agent_id"])  # 传入智能体ID用于记忆系统权限验证
        )

        self._update_objective_interest()
        self._is_initialized = True

    @staticmethod
    def calculate_power_tiers_for_all(agents: List["BaseAgent"]) -> None:
        """
        批量计算所有智能体的实力层级

        Args:
            agents: 所有智能体列表
        """
        # 1. 收集所有智能体的PowerMetrics
        power_metrics_list = [agent._init_config["power_metrics"] for agent in agents]

        # 2. 使用PowerTierClassifier批量分类
        tiers = PowerTierClassifier.classify_all(power_metrics_list)

        # 3. 设置每个智能体的power_tier
        for agent, tier in zip(agents, tiers):
            agent.power_tier = tier

    @classmethod
    def batch_initialize(cls, agents_config: List[Dict]) -> List["BaseAgent"]:
        """
        批量初始化智能体的推荐方法

        遵循正确的初始化顺序：
        1. 创建所有智能体基本实例
        2. 批量计算所有智能体的实力层级
        3. 根据层级配置领导类型
        4. 完成所有智能体的最终初始化

        Args:
            agents_config: 智能体配置列表

        Returns:
            完整初始化后的智能体列表
        """
        # 步骤1: 创建所有智能体基本实例
        agents = []
        for config in agents_config:
            agent = cls(
                agent_id=config["agent_id"],
                name=config["name"],
                region=config["region"],
                power_metrics=config["power_metrics"]
            )
            agents.append(agent)

        # 步骤2: 批量计算所有智能体的实力层级
        cls.calculate_power_tiers_for_all(agents)

        # 步骤3: 为需要的智能体配置领导类型
        for config, agent in zip(agents_config, agents):
            if agent.power_tier in [PowerTier.SUPERPOWER, PowerTier.GREAT_POWER]:
                if "leader_type" not in config or config["leader_type"] is None:
                    raise ValueError(
                        f"{config['name']}被系统判定为超级大国/大国，必须配置领导类型"
                    )
                agent.set_leader_type(config["leader_type"])
            else:
                # 中等强国和小国不需要配置leader_type
                agent.set_leader_type(None)

        # 步骤4: 完成所有智能体的最终初始化
        for agent in agents:
            agent.complete_initialization()

        return agents

"""
```

```python
from typing import Dict, List, Optional
from dataclasses import dataclass
from enum import Enum

class MetricType(str, Enum):
    """指标类型"""
    INDEPENDENT = "independent"      # 自变量
    INTERMEDIARY = "intermediary"    # 中介变量
    ENVIRONMENT = "environment"      # 体系环境指标
    DEPENDENT = "dependent"          # 因变量

@dataclass
class Metric:
    """指标数据类"""
    metric_type: MetricType
    name: str
    value: float
    unit: str = ""
    metadata: Optional[Dict] = None

class MetricsCalculator:
    """指标计算器"""

    @staticmethod
    def calculate_independent_metrics(
        agent_states: List[Dict]
    ) -> List[Metric]:
        """

        Args:
            agent_states: 智能体状态列表

        Returns:
            自变量指标列表
        """
        metrics = []

        for agent in agent_states:
            agent_id = agent.get("agent_id", "")
            power_metrics = agent.get("power_metrics", {})

            # 综合实力指数
            metrics.append(Metric(
                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_comprehensive_power",
                value=agent.get("comprehensive_power", 0),
                unit="分"
            ))

            # 克莱因方程五要素指标
            metrics.append(Metric(
                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_critical_mass",
                value=power_metrics.get("critical_mass", 0),
                unit="分"
            ))

            metrics.append(Metric(
                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_economic_capability",
                value=power_metrics.get("economic_capability", 0),
                unit="分"
            ))

            metrics.append(Metric(
                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_military_capability",
                value=power_metrics.get("military_capability", 0),
                unit="分"
            ))

            metrics.append(Metric(
                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_strategic_purpose",
                value=power_metrics.get("strategic_purpose", 0),
                unit="分"
            ))

                power_metrics.get("strategic_purpose", 0) +

                metric_type=MetricType.INDEPENDENT,
                name=f"{agent_id}_material_power",
                value=material_power,
                unit="分"

                metric_type=MetricType.INDEPENDENT,

        if not interactions:
            return metrics

        # 战略行为特征指数
        total_interactions = len(interactions)
        multilateral_cooperations = sum(
            1 for i in interactions
            if "multilateral" in i.get("interaction_type", "").lower()
        )
        unilateral_actions = sum(
            1 for i in interactions
            if "unilateral" in i.get("interaction_type", "").lower()
        )

        if total_interactions > 0:
            multilateral_preference = (
                multilateral_cooperations / total_interactions * 100
            )
            unilateral_ratio = unilateral_actions / total_interactions * 100
        else:
            multilateral_preference = 0.0
            unilateral_ratio = 0.0

        metrics.append(Metric(
            metric_type=MetricType.INTERMEDIARY,
            name="multilateral_cooperation_preference",
            value=multilateral_preference,
            unit="%"
        ))

        metrics.append(Metric(
            metric_type=MetricType.INTERMEDIARY,
            name="unilateral_action_ratio",
            value=unilateral_ratio,
            unit="%"
        ))

        # 有效联盟数量
        alliance_count = MetricsCalculator._calculate_alliance_count(interactions)
        metrics.append(Metric(
            metric_type=MetricType.INTERMEDIARY,
            name="effective_alliance_count",
            value=float(alliance_count),
            unit="个"
        )

        return metrics

    @staticmethod
    def calculate_environment_metrics(
        norms: List[Dict],
        conflicts: List[Dict],
        agent_states: Optional[List[Dict]] = None
    ) -> List[Metric]:
        """
        计算体系环境指标 - 对应方案6.3节

        Args:
            norms: 国际规范列表
            conflicts: 冲突记录列表
            agent_states: 智能体状态列表（可选，用于计算制度化程度）

        Returns:
            体系环境指标列表
        """
        metrics = []

        # 国际规范有效性指数
        if norms:
            avg_validity = sum(
                n.get("validity", 0) for n in norms
            ) / len(norms)
            avg_adherence = sum(
                n.get("adherence_rate", 0) for n in norms
            ) / len(norms)
            norm_effectiveness = (avg_validity * avg_adherence) / 100
        else:
            norm_effectiveness = 0.0

        metrics.append(Metric(
            metric_type=MetricType.ENVIRONMENT,
            name="international_norm_effectiveness",
            value=norm_effectiveness,
            unit="分"
        ))

        # 全球冲突水平指数
        if conflicts:
            conflict_levels = [
                c.get("impact_level", 0) * 100 for c in conflicts
            ]
            global_conflict_level = sum(conflict_levels) / len(conflicts)
        else:
            global_conflict_level = 0.0

        metrics.append(Metric(
            metric_type=MetricType.ENVIRONMENT,
            name="global_conflict_level",
            value=global_conflict_level,
            unit="分"
        ))

        # 制度化程度指数
        if agent_states:
            institutionalization = MetricsCalculator._calculate_institutionalization(
                norms, agent_states
            )
            metrics.append(Metric(
                metric_type=MetricType.ENVIRONMENT,
                name="institutionalization_level",
                value=institutionalization,
                unit="分"
            ))

        return metrics

    @staticmethod
    def calculate_dependent_metrics(
        agent_powers: List[float],
        power_distribution: Dict[str, float],
        norm_effectiveness: float,
        conflict_level: float,
        alliance_count: int,
        institutionalization: float
    ) -> List[Metric]:
        """
        计算因变量指标 - 对应方案6.4节

        Args:
            agent_powers: 各国实力列表
            power_distribution: 实力分布
            norm_effectiveness: 国际规范有效性指数 (0-100)
            conflict_level: 全球冲突水平指数 (0-100)
            alliance_count: 有效联盟数量
            institutionalization: 制度化程度指数 (0-100)

        Returns:
            因变量指标列表
        """
        metrics = []

        if not agent_powers:
            return metrics

        # 实力集中度指数
        total_power = sum(agent_powers)
        if total_power > 0:
            top3_power = sum(sorted(agent_powers, reverse=True)[:3])
            power_concentration = (top3_power / total_power) * 100
        else:
            power_concentration = 0.0

        metrics.append(Metric(
            metric_type=MetricType.DEPENDENT,
            name="power_concentration_index",
            value=power_concentration,
            unit="%"
        ))

        # 判定国际格局类型（权力结构）
        if power_concentration > 60:
            pattern_type = "unipolar"
        elif power_concentration > 40:
            pattern_type = "bipolar"
        elif power_concentration > 25:
            pattern_type = "multipolar"
        else:
            pattern_type = "fragmented"

        # 判定国际秩序类型 - 导入并使用完整判定逻辑
        from config.order_types import determine_order_type

        order_type = determine_order_type(
            power_concentration=power_concentration,
            norm_effectiveness=norm_effectiveness,
            conflict_level=conflict_level,
            alliance_count=alliance_count,
            institutionalization=institutionalization
        )

        # 获取秩序类型配置
        from config.order_types import ORDER_TYPES_CONFIG, InternationalOrderType
        order_config = ORDER_TYPES_CONFIG.get(InternationalOrderType(order_type))

        # 添加国际格局类型指标
        metrics.append(Metric(
            metric_type=MetricType.DEPENDENT,
            name="international_power_pattern",
            value=0.0,  # 使用元数据存储实际值
            unit="",
            metadata={"pattern_type": pattern_type}
        ))

        # 添加国际秩序类型指标
        metrics.append(Metric(
            metric_type=MetricType.DEPENDENT,
            name="international_order_type",
            value=0.0,  # 使用元数据存储实际值
            unit="",
            metadata={
                "order_type": order_type,
                "governance_model": order_config.governance_model if order_config else "",
                "interaction_characteristics": order_config.interaction_characteristics if order_config else [],
                "historical_examples": order_config.historical_examples if order_config else []
            }
        ))

        return metrics

    @staticmethod
    def calculate_all_metrics(
        agent_states: List[Dict],
        interactions: List[Dict],
        norms: List[Dict],
        conflicts: List[Dict]
    ) -> Dict[str, List[Metric]]:
        """
        计算所有指标

        Args:
            agent_states: 智能体状态列表
            interactions: 互动记录列表
            norms: 国际规范列表
            conflicts: 冲突记录列表

        Returns:
            所有指标的字典
        """
        agent_powers = [a.get("comprehensive_power", 0) for a in agent_states]

        # 计算体系环境指标
        environment_metrics = MetricsCalculator.calculate_environment_metrics(norms, conflicts, agent_states)

        # 计算联盟数量指标
        alliance_count = MetricsCalculator._calculate_alliance_count(interactions)

        # 计算制度化程度指标
        institutionalization = MetricsCalculator._calculate_institutionalization(norms, agent_states)

        # 提取环境指标值用于因变量计算
        norm_effectiveness = 50.0
        conflict_level = 20.0

        for metric in environment_metrics:
            if metric.name == "international_norm_effectiveness":
                norm_effectiveness = metric.value
            elif metric.name == "global_conflict_level":
                conflict_level = metric.value

        return {
            "independent": MetricsCalculator.calculate_independent_metrics(agent_states),
            "intermediary": MetricsCalculator.calculate_intermediary_metrics(interactions),
            "environment": environment_metrics,
            "dependent": MetricsCalculator.calculate_dependent_metrics(
                agent_powers=agent_powers,
                power_distribution={},
                norm_effectiveness=norm_effectiveness,
                conflict_level=conflict_level,
                alliance_count=alliance_count,
                institutionalization=institutionalization
            )
        }

    @staticmethod
    def _calculate_alliance_count(interactions: List[Dict]) -> int:
        """
        计算有效联盟数量

        判定标准：
        - 军事同盟或经济同盟
        - 状态为活跃的联盟
        - 参与方数量≥2的联盟

        Args:
            interactions: 互动记录列表

        Returns:
            有效联盟数量
        """
        # 统计活跃的军事和经济同盟数量
        active_alliances = set()
        for interaction in interactions:
            if (interaction.get("interaction_type") in ["military_alliance", "economic_alliance"] and
                interaction.get("status") == "active"):
                # 使用联盟ID或参与方组合作为唯一标识
                alliance_key = interaction.get("alliance_id",
                    "-".join(sorted(interaction.get("participants", []))))
                active_alliances.add(alliance_key)

        return len(active_alliances)

    @staticmethod
    def _calculate_institutionalization(norms: List[Dict], agent_states: List[Dict]) -> float:
        """
        计算制度化程度指数（0-100分）

        计算方法：
        1. 国际规范覆盖度：规范数量×平均遵守率
        2. 国际组织参与度：活跃国际组织数量×平均参与度
        3. 多边机制运作效率：多边互动占比×互动成功率

        Args:
            norms: 国际规范列表
            agent_states: 智能体状态列表

        Returns:
            制度化程度指数
        """
        # 1. 国际规范覆盖度（权重40%）
        norm_coverage = 0.0
        if norms:
            avg_adherence = sum(n.get("adherence_rate", 0) for n in norms) / len(norms)
            norm_coverage = len(norms) * avg_adherence / 10  # 规范数量×平均遵守率

        # 2. 国际组织参与度（权重30%）
        org_participation = 0.0
        active_agents = len([a for a in agent_states if a.get("comprehensive_power", 0) > 0])
        if active_agents > 0:
            avg_org_count = sum(len(a.get("international_organizations", [])) for a in agent_states) / active_agents
            org_participation = avg_org_count * 10  # 每个智能体平均参与的国际组织数量

        # 3. 多边机制运作效率（权重30%）
        multilateral_efficiency = 0.0
        # 从智能体状态中获取多边互动信息
        total_multilateral = 0
        successful_multilateral = 0
        for agent in agent_states:
            # 假设智能体状态中包含多边互动统计
            total_multilateral += agent.get("multilateral_interactions", 0)
            successful_multilateral += agent.get("successful_multilateral_interactions", 0)

        if total_multilateral > 0:
            multilateral_efficiency = (successful_multilateral / total_multilateral) * 100

        # 综合计算制度化程度指数
        institutionalization = (norm_coverage * 0.4 +
                               org_participation * 0.3 +
                               multilateral_efficiency * 0.3)

        # 确保值在0-100范围内
        return max(0.0, min(100.0, institutionalization))
```

def determine_order_type(
    power_concentration: float,
    norm_effectiveness: float,
    conflict_level: float,
    alliance_count: int,
    institutionalization: float
) -> InternationalOrderType:
    """
    判定国际秩序类型 - 综合多项指标进行判定

    Args:
        power_concentration: 实力集中度指数 (0-100)
        norm_effectiveness: 国际规范有效性指数 (0-100)
        conflict_level: 全球冲突水平指数 (0-100)
        alliance_count: 有效联盟数量
        institutionalization: 制度化程度指数 (0-100)

    Returns:
        国际秩序类型
    """
    # 无秩序型判定：高冲突、低规范有效性
    if conflict_level > 60 and norm_effectiveness < 30:
        return InternationalOrderType.DISORDER

    # 霸权秩序判定：高实力集中度、中等规范有效性
    if power_concentration > 60 and norm_effectiveness > 30:
        return InternationalOrderType.HEGEMONIC

    # 规则/制度秩序判定：高规范有效性、高制度化程度
    if norm_effectiveness > 60 and institutionalization > 60:
        return InternationalOrderType.RULE_BASED

    # 均势秩序判定：中等实力集中度、一定联盟数量、中等规范有效性
    if (30 < power_concentration < 60 and
        alliance_count >= 2 and
        norm_effectiveness > 30):
        return InternationalOrderType.BALANCE_OF_POWER

    # 混合型秩序判定：多个条件部分满足
    if (norm_effectiveness > 40 and
        institutionalization > 30 and
        40 < power_concentration < 70):
        return InternationalOrderType.MIXED

    # 默认返回无秩序型
    return InternationalOrderType.DISORDER

### 5.1 决策生成流程（增强版，包含决策理由记录）

```python
"""
决策生成流程 - 整合LLM引擎、规则校验与增强日志记录
"""
from typing import Dict, List, Optional
from core.llm_engine import LLMEngine
from core.validator import RuleValidator, ValidationResult
from core.prompt_engine import PromptEngine
from core.storage import StorageEngine
from core.logger import EnhancedLogger, DecisionReasoning, ActionRecord, SpeechRecord

class DecisionEngine:
    """决策引擎（增强版，包含理由记录）"""

    def __init__(
        self,
        llm_engine: LLMEngine,
        validator: RuleValidator,
        storage: StorageEngine,
        logger: EnhancedLogger
    ):
        self.llm_engine = llm_engine
        self.validator = validator
        self.storage = storage
        self.logger = logger
        self.max_retries = 3

    async def generate_decision_for_agent(
        self,
        agent: 'BaseAgent',
        environment_state: Dict,
        simulation_id: str,
        round: int,
        use_rotation: bool = True  # 是否使用API-key轮替调用
    ) -> Dict:
        """
        为单个智能体生成决策（包含完整理由记录）

        Args:
            agent: 智能体实例
            environment_state: 环境状态
            simulation_id: 仿真ID
            round: 当前轮次

        Returns:
            决策结果（包含决策理由）
        """
        available_functions = agent.get_available_functions()
        prohibited_functions = agent.get_prohibited_functions()

        # 处理leader_type可能为None的情况（中等强国和小国）
        leader_type_str = agent.state.leader_type.value if agent.state.leader_type else "none"

        prompt = PromptEngine.build_agent_prompt(
            agent_name=agent.state.name,
            agent_type=agent.state.agent_type,
            power_tier=agent.state.power_tier.value,
            objective_interest=getattr(agent.state, 'objective_interest', ''),
            leader_type=leader_type_str,
            environment_state=environment_state,
            recent_history=agent.state.memory.recent_decisions,
            current_events=environment_state.get('active_events', [])
        )

        for attempt in range(self.max_retries):
            # 1. 调用LLM生成决策（包含理由）
            llm_result = await self.llm_engine.make_decision(
                agent_id=agent.state.agent_id,
                prompt=prompt,
                available_functions=available_functions,
                prohibited_functions=prohibited_functions,
                use_rotation=use_rotation
            )

            # 2. 验证决策
            validation_result = self.validator.validate_decision(
                leader_type=agent.state.leader_type,
                function_name=llm_result.get('function', ''),
                function_args=llm_result.get('arguments', {}),
                objective_interest=getattr(agent.state, 'objective_interest', ''),
                current_power=agent.state.power_metrics.calculate_comprehensive_power()
            )

            # 3. 检查验证结果
            if validation_result.is_valid:
                # 决策有效
                function_name = llm_result.get('function', '')
                function_args = llm_result.get('arguments', {})

                # 创建决策理由记录
                reasoning = DecisionReasoning(
                    decision_id=f"decision_{agent.state.agent_id}_{round}_{attempt}",
                    agent_id=agent.state.agent_id,
                    round=round,
                    situation_analysis=llm_result.get('reasoning', {}).get('situation_analysis', ''),
                    strategic_consideration=llm_result.get('reasoning', {}).get('strategic_consideration', ''),
                    expected_outcome=llm_result.get('reasoning', {}).get('expected_outcome', ''),
                    alternatives=llm_result.get('reasoning', {}).get('alternatives', []),
                    full_reasoning=llm_result.get('reasoning', {}).get('full_reasoning', '')
                )

                decision = {
                    'agent_id': agent.state.agent_id,
                    'function': function_name,
                    'arguments': function_args,
                    'validation': {'is_valid': True},
                    'reasoning': reasoning,
                    'attempt': attempt + 1
                }

                # 更新智能体状态
                agent.record_decision(
                    function_name,
                    function_args
                )

                # 记录决策日志（包含理由）
                self.logger.log_decision(
                    simulation_id=simulation_id,
                    round=round,
                    agent_id=agent.state.agent_id,
                    decision_type=function_name,
                    decision_content=str(function_args),
                    reasoning=reasoning,
                    metadata={'attempt': attempt + 1}
                )

                # 保存决策记录（包含理由）
                self.storage.save_decision(
                    simulation_id=simulation_id,
                    round=round,
                    agent_id=agent.state.agent_id,
                    function_name=function_name,
                    function_args=function_args,
                    validation_result={'is_valid': True},
                    reasoning={
                        'situation_analysis': reasoning.situation_analysis,
                        'strategic_consideration': reasoning.strategic_consideration,
                        'expected_outcome': reasoning.expected_outcome,
                        'alternatives': reasoning.alternatives,
                        'full_reasoning': reasoning.full_reasoning
                    }
                )

                return decision
            else:
                # 决策无效，需要重试
                if attempt == self.max_retries - 1:
                    # 最后一次尝试失败，返回错误
                    return {
                        'agent_id': agent.state.agent_id,
                        'function': None,
                        'error': validation_result.error_message,
                        'validation': {
                            'is_valid': False,
                            'error': validation_result.error_message
                        }
                    }

        return {
            'agent_id': agent.state.agent_id,
            'function': None,
            'error': 'Max retries exceeded'
        }

    async def generate_decisions_for_all_agents(
        self,
        agents: List['BaseAgent'],
        environment_state: Dict,
        simulation_id: str,
        round: int,
        use_rotation: bool = False  # 步骤2并行决策时不使用轮替
    ) -> List[Dict]:
        """
        并行生成所有智能体的决策（步骤2）

        API-key使用策略：
        - use_rotation=False（步骤2）：多个API-key同时调用，每个智能体使用独立key
        - use_rotation=True（其他步骤）：API-key轮替调用

        Args:
            agents: 智能体列表
            environment_state: 环境状态
            simulation_id: 仿真ID
            round: 当前轮次
            use_rotation: 是否使用API-key轮替调用

        Returns:
            决策结果列表
        """
        import asyncio

        tasks = [
            self.generate_decision_for_agent(
                agent=agent,
                environment_state=environment_state,
                simulation_id=simulation_id,
                round=round,
                use_rotation=use_rotation
            )
            for agent in agents
        ]

        return await asyncio.gather(*tasks)
```

#### 6.1.3 并行决策架构

```python
"""
多智能体并行决策机制
"""
import asyncio
from typing import Dict, List, Optional, Set
from dataclasses import dataclass

@dataclass
class DecisionRequest:
    """决策请求"""
    agent_id: str
    agent_type: str
    prompt: str
    available_functions: List[Dict]
    prohibited_functions: Set[str]
    # 依赖关系
    depends_on: Optional[List[str]] = None
    # 决策结果
    result: Optional[Dict] = None
    # 状态
    is_completed: bool = False
    is_failed: bool = False
    error: Optional[str] = None

class ParallelDecisionMaker:
    """并行决策管理器"""

    def __init__(self, llm_engine):
        self.llm_engine = llm_engine
        self._lock = asyncio.Lock()

    async def make_parallel_decisions(
        self,
        requests: List[DecisionRequest]
    ) -> Dict[str, Dict]:
        """
        并行执行决策请求

        Args:
            requests: 决策请求列表

        Returns:
            智能体ID到决策结果的映射
        """
        # 1. 构建依赖图
        dependency_map = self._build_dependency_map(requests)

        # 2. 执行拓扑排序
        execution_order = self._topological_sort(requests, dependency_map)

        # 3. 按顺序执行（无依赖的可并行）
        for stage in execution_order:
            tasks = self._prepare_stage_tasks(stage, requests)
            await asyncio.gather(*tasks)

        # 4. 返回结果
        return {
            req.agent_id: req.result
            for req in requests
            if req.is_completed and req.result
        }

    def _build_dependency_map(
        self,
        requests: List[DecisionRequest]
    ) -> Dict[str, Set[str]]:
        """构建依赖映射"""
        return {
            req.agent_id: set(req.depends_on or [])
            for req in requests
        }

    def _topological_sort(
        self,
        requests: List[DecisionRequest],
        dependency_map: Dict[str, Set[str]]
    ) -> List[List[str]]:
        """拓扑排序，返回执行阶段"""
        result = []
        remaining = {req.agent_id for req in requests}

        while remaining:
            # 找出无依赖的任务
            stage = [
                agent_id
                for agent_id in remaining
                if not dependency_map[agent_id] & remaining
            ]

            if not stage:
                # 循环依赖，全部执行
                stage = list(remaining)

            result.append(stage)
            remaining -= set(stage)

        return result

    async def _prepare_stage_tasks(
        self,
        stage: List[str],
        requests: List[DecisionRequest]
    ) -> List:
        """准备阶段任务"""
        tasks = []
        request_map = {req.agent_id: req for req in requests}

        for agent_id in stage:
            req = request_map[agent_id]

            async def _execute(request):
                try:
                    result = await self.llm_engine.make_decision(
                        agent_id=request.agent_id,
                        prompt=request.prompt,
                        available_functions=request.available_functions,
                        prohibited_functions=request.prohibited_functions
                    )
                    request.result = result
                    request.is_completed = True
                except Exception as e:
                    request.is_failed = True
                    request.error = str(e)

            tasks.append(_execute(req))

        return tasks
```

```python
"""
FastAPI 后端应用 - 核心API服务
"""
from fastapi import FastAPI, WebSocket, WebSocketDisconnect
from fastapi.middleware.cors import CORSMiddleware
from fastapi.staticfiles import StaticFiles
from typing import List
import uvicorn

# 创建FastAPI应用实例
app = FastAPI(
    title="道义现实主义社会模拟仿真系统 API",
    description="基于LLM的社会模拟仿真系统后端API",
    version="1.0.0",
    docs_url="/api/docs",
    redoc_url="/api/redoc"
)

# CORS 配置
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# WebSocket 管理器
class ConnectionManager:
    """WebSocket连接管理器"""

    def __init__(self):
        self.active_connections: List[WebSocket] = []

    async def connect(self, websocket: WebSocket):
        await websocket.accept()
        self.active_connections.append(websocket)

    def disconnect(self, websocket: WebSocket):
        self.active_connections.remove(websocket)

    async def send_personal_message(self, message: dict, websocket: WebSocket):
        await websocket.send_json(message)

    async def broadcast(self, message: dict):
        for connection in self.active_connections:
            try:
                await connection.send_json(message)
            except:
                self.disconnect(connection)

manager = ConnectionManager()

# 导入并注册API路由
from backend.api.simulation import router as simulation_router
from backend.api.agents import router as agents_router
from backend.api.events import router as events_router
from backend.api.data import router as data_router
from backend.api.export import router as export_router

app.include_router(simulation_router, prefix="/api/simulation", tags=["仿真管理"])
app.include_router(agents_router, prefix="/api/agents", tags=["智能体管理"])
app.include_router(events_router, prefix="/api/events", tags=["事件管理"])
app.include_router(data_router, prefix="/api/data", tags=["数据查询"])
app.include_router(export_router, prefix="/api/export", tags=["结果导出"])

# WebSocket 端点
@app.websocket("/ws/simulation")
async def websocket_endpoint(websocket: WebSocket):
    """仿真状态实时推送"""
    await manager.connect(websocket)
    try:
        while True:
            # 这里可以添加接收客户端消息的逻辑
            await websocket.receive_text()
    except WebSocketDisconnect:
        manager.disconnect(websocket)

# 挂载前端静态文件（生产部署时使用）
try:
    app.mount("/", StaticFiles(directory="frontend/build", html=True), name="frontend")
except Exception:
    print("前端静态文件未构建，开发模式请使用前端开发服务器")

if __name__ == "__main__":
    # 开发模式启动
    uvicorn.run(
        "backend.main:app",
        host="0.0.0.0",
        port=8000,
        reload=True,
        log_level="info"
    )
```

#### 8.3.3 状态管理
```typescript
// frontend/src/store/index.ts - 全局状态管理
import React, { createContext, useContext, useReducer, ReactNode } from 'react';
import { SimulationState } from '../services/simulation';

interface AppState {
  simulation: {
    state: SimulationState;
    isLoading: boolean;
    error: string | null;
  };
  agents: {
    list: Agent[];
    isLoading: boolean;
    error: string | null;
  };
}

const initialState: AppState = {
  simulation: {
    state: {
      current_round: 0,
      active_events: 0,
      power_pattern: '未判定',
      order_type: '未判定',
      is_running: false,
      is_paused: false,
    },
    isLoading: false,
    error: null,
  },
  agents: {
    list: [],
    isLoading: false,
    error: null,
  },
};

type Action =
  | { type: 'SET_SIMULATION_STATE'; payload: SimulationState }
  | { type: 'SET_SIMULATION_LOADING'; payload: boolean }
  | { type: 'SET_SIMULATION_ERROR'; payload: string | null }
  | { type: 'SET_AGENTS'; payload: Agent[] }
  | { type: 'SET_AGENTS_LOADING'; payload: boolean }
  | { type: 'SET_AGENTS_ERROR'; payload: string | null };

function reducer(state: AppState, action: Action): AppState {
  switch (action.type) {
    case 'SET_SIMULATION_STATE':
      return {
        ...state,
        simulation: {
          ...state.simulation,
          state: action.payload,
        },
      };
    case 'SET_SIMULATION_LOADING':
      return {
        ...state,
        simulation: {
          ...state.simulation,
          isLoading: action.payload,
        },
      };
    case 'SET_SIMULATION_ERROR':
      return {
        ...state,
        simulation: {
          ...state.simulation,
          error: action.payload,
        },
      };
    case 'SET_AGENTS':
      return {
        ...state,
        agents: {
          ...state.agents,
          list: action.payload,
        },
      };
    case 'SET_AGENTS_LOADING':
      return {
        ...state,
        agents: {
          ...state.agents,
          isLoading: action.payload,
        },
      };
    case 'SET_AGENTS_ERROR':
      return {
        ...state,
        agents: {
          ...state.agents,
          error: action.payload,
        },
      };
    default:
      return state;
  }
}

const AppContext = createContext<{
  state: AppState;
  dispatch: React.Dispatch<Action>;
}>({
  state: initialState,
  dispatch: () => null,
});

export function Provider({ children }: { children: ReactNode }) {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <AppContext.Provider value={{ state, dispatch }}>
      {children}
    </AppContext.Provider>
  );
}

export function useAppState() {
  return useContext(AppContext);
}
```

### 9.2 主要风险分析

| 风险点 | 影响等级 | 应对措施 |
|-------|---------|---------|
| LLM输出不稳定 | 高 | 函数调用约束 + 多次重试 + 规则校验 |

### 9.3 性能优化策略

1. **LLM调用优化**

   - 使用本地模型降低成本

   - 合理划分任务批次
   - 避免过多并发连接

3. **数据存储优化**
   - 批量写入减少事务次数
   - 定期归档历史数据

### C. 快速启动脚本

```python
# main.py
import asyncio
from workflows.multi_round import MultiRoundWorkflow
from workflows.single_round import SingleRoundWorkflow
from core.environment import EnvironmentEngine
from core.storage import StorageEngine
from core.llm_engine import LLMEngine, SiliconFlowProvider
from core.validator import RuleValidator
from observation.metrics import MetricsCalculator
from decision_engine import DecisionEngine
from entities.state_agent import StateAgent
from entities.power_system import PowerMetrics
from core.validator import LeaderType
from config.settings import SimulationConfig

async def main():
    # 初始化配置
    config = SimulationConfig()

    # 初始化引擎 - 使用SiliconFlow平台的DeepSeek-V3.2模型
    # 支持多个API-key用于并行调用和速率限制规避
    if not config.llm_api_keys:
        raise ValueError("请配置至少一个SiliconFlow API Key")
    llm_provider = SiliconFlowProvider(
        api_key=config.llm_api_keys,  # 支持多个API-key列表
        base_url=config.llm_base_url,
        model=config.llm_model
    )
    llm_engine = LLMEngine(llm_provider)
    validator = RuleValidator()
    storage = StorageEngine(config.database.database_path)
    logger = EnhancedLogger("logs")  # 初始化增强日志记录器
    environment = EnvironmentEngine()
    metrics_calculator = MetricsCalculator()

    # 初始化决策引擎（包含增强日志记录）
    decision_engine = DecisionEngine(llm_engine, validator, storage, logger)

    # 初始化工作流（包含增强日志记录）
    single_round_workflow = SingleRoundWorkflow(
        environment, decision_engine, metrics_calculator, storage, logger
    )
    multi_round_workflow = MultiRoundWorkflow(single_round_workflow, storage)

    # 创建智能体
    # 克莱因方程示例：
    # 物质要素（硬实力）：C(0-100) + E(0-200) + M(0-200)
    # 精神要素（软实力）：S(0.5-1) + W(0.5-1)
    # 综合国力 = (C+E+M) × (S+W)，最大1000分

    # 准备所有智能体的实力指标
    power_metrics_list = [
        PowerMetrics(
            critical_mass=85,        # C - 基本实体 0-100分
            economic_capability=180,   # E - 经济实力 0-200分
            military_capability=170,   # M - 军事实力 0-200分
            strategic_purpose=0.9,    # S - 战略目标 0.5-1分
            national_will=0.95        # W - 国家意志 0.5-1分
        ),
        PowerMetrics(
            critical_mass=75,        # C - 基本实体 0-100分
            economic_capability=150,   # E - 经济实力 0-200分
            military_capability=130,   # M - 军事实力 0-200分
            strategic_purpose=0.75,    # S - 战略目标 0.5-1分
            national_will=0.8        # W - 国家意志 0.5-1分
        ),
        PowerMetrics(
            critical_mass=35,        # C - 基本实体 0-100分
            economic_capability=60,   # E - 经济实力 0-200分
            military_capability=40,   # M - 军事实力 0-200分
            strategic_purpose=0.6,   # S - 战略目标 0.5-1分
            national_will=0.65       # W - 国家意志 0.5-1分
        )
    ]

    # 使用正态分布方法批量分类实力层级
    tiers = PowerTierClassifier.classify_all(power_metrics_list)

    # 创建智能体（临时不初始化状态）
    agent1 = StateAgent(
        agent_id="country_a",
        name="国家A",
        region="区域1",
        power_metrics=power_metrics_list[0],
        leader_type=LeaderType.BAQUAN
    )

    agent2 = StateAgent(
        agent_id="country_b",
        name="国家B",
        region="区域2",
        power_metrics=power_metrics_list[1],
        leader_type=LeaderType.WANGDAO
    )

    # 注意：小国不需要leader_type参数
    agent3 = SmallStateAgent(
        agent_id="country_c",
        name="国家C",
        region="区域1",
        power_metrics=power_metrics_list[2]
    )

    # 完成所有智能体的初始化（分配实力层级）
    agents = [agent1, agent2, agent3]
    for agent, tier in zip(agents, tiers):
        agent.complete_init(tier)

    # 验证分类结果
    print("实力层级分类结果：")
    for agent, pm in zip(agents, power_metrics_list):
        power = pm.calculate_comprehensive_power()
        print(f"{agent.state.name}: 综合国力={power:.1f}, 层级={agent.state.power_tier.value}")

    # 保存仿真开始
    simulation_id = "sim_001"
    storage.save_simulation_start(simulation_id, config.model_dump())

    # 运行仿真
    results = await multi_round_workflow.execute(
        agents=agents,
        simulation_id=simulation_id,
        total_rounds=config.total_rounds
    )

    # 保存仿真结束
    storage.save_simulation_end(
        simulation_id,
        len(results),
        status="completed"
    )

    print(f"仿真完成，共执行{len(results)}轮")

if __name__ == "__main__":
    asyncio.run(main())
```

## 十一、SiliconFlow API配置指南

### 11.1 获取API密钥

1. 访问 [SiliconFlow官网](https://cloud.siliconflow.cn/)
2. 注册并登录账号
3. 在控制台创建API密钥
4. 复制API密钥备用

### 11.2 配置文件设置

创建 `config/config.yaml` 文件：

```yaml
# SiliconFlow API配置
llm_provider: "siliconflow"
llm_model: "deepseek-ai/DeepSeek-V3.2"
llm_api_key: "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx"  # 替换为你的API密钥
llm_base_url: "https://api.siliconflow.cn/v1"
llm_temperature: 0.7
llm_max_tokens: 4096

# 仿真参数配置
total_rounds: 100
round_duration_months: 6
leader_term_rounds: 4

# 事件配置
random_event_probability: 0.1
enable_user_events: true

# 存储配置
database_path: "data/database.db"
auto_save_interval: 5
```

### 11.3 环境变量配置（推荐）

```bash
# Windows PowerShell
$env:SILICONFLOW_API_KEYS = "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx1,sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx2"

# Linux/Mac
export SILICONFLOW_API_KEYS="sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx1,sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxx2"
```

然后在代码中使用：

```python
import os
from config.settings import SimulationConfig

# 从环境变量读取多个API-key（逗号分隔）
api_keys_str = os.getenv("SILICONFLOW_API_KEYS", "")
api_keys = [key.strip() for key in api_keys_str.split(",") if key.strip()]

config = SimulationConfig(
    llm_api_keys=api_keys if api_keys else config.llm_api_keys
)
```

## 十二、增强日志系统使用指南

### 12.1 日志类型说明

| 日志类型 | 英文标识 | 说明 | 记录内容 |
|---------|---------|------|---------|
| 决策日志 | DECISION | 记录所有智能体决策 | 决策类型、决策内容、决策理由（局势分析、战略考量、预期结果、替代方案） |
| 行动日志 | ACTION | 记录所有执行的行动 | 行动类型、目标对象、行动参数、行动理由、影响评估 |
| 发言日志 | SPEECH | 记录所有公开声明和发言 | 发言类型、发言内容、发言理由、受众群体、反响记录 |
| 判断日志 | JUDGMENT | 记录所有判断和评估 | 判断类型、判断结果、判断理由、判断上下文 |
| 统计日志 | METRIC | 记录数据统计和趋势分析 | 统计类别、指标数值、趋势描述、分析结果、变动原因 |
| 事件日志 | EVENT | 记录事件触发 | 事件类型、事件描述、参与者、影响级别 |
| 错误日志 | ERROR | 记录系统错误 | 错误类型、错误信息、错误堆栈 |
| 调试日志 | DEBUG | 记录调试信息 | 调试内容、变量状态、执行流程 |
