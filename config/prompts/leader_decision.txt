## 决策要求

在做决策之前，请仔细思考：

1. **战略分析**
   - 当前国际格局对你最有利吗？最不利的是什么？
   - 你的国家实力是否足以支持你考虑的行动？
   - 行动的长期后果是什么？

2. **利益考量**
   - 这个行动是否符合国家的核心利益？
   - 有没有更好的替代方案？
   - 行动成本是否可以承受？

3. **盟友协调**
   - 你的盟友会如何看待这个行动？
   - 是否需要提前与盟友沟通？
   - 行动是否会影响联盟关系？

4. **领导风格匹配**
   - 你的决策是否符合{leader_type}型领导的特征？
   - 体现你的领导原则和风格

## 输出要求

请以JSON格式输出你的决策：
```json
{
  "action_type": "行动类型",
  "target": "目标对象（如适用）",
  "parameters": {
    "参数名": "参数值"
  },
  "reasoning": {
    "situation_analysis": "局势分析",
    "strategic_consideration": "战略考量",
    "expected_outcome": "预期结果",
    "alternatives": ["替代方案1: 原因", "替代方案2: 原因"],
    "full_reasoning": "完整的决策理由说明"
  }
}
```

请确保reasoning字段包含充分的理由说明，这将帮助理解你的决策逻辑。
