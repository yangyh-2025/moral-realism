# Frontend - 国际秩序ABM仿真系统

## 技术栈

- Vue 3.4+
- Vite 5.0+
- Element Plus
- ECharts 5.5+
- Axios
- Socket.IO Client
- Pinia

## 安装依赖

\`\`\`bash
npm install
\`\`\`

## 开发运行

\`\`\`bash
npm run dev
\`\`\`

访问: http://localhost:3000

## 构建

\`\`\`bash
npm run build
\`\`\`

## 代码规范

\`\`\`bash
npm run lint
\`\`\`

## 目录结构

- `src/` - 源代码目录
  - `api/` - API客户端模块
  - `components/` - 共享组件
  - `store/` - Pinia状态管理
  - `views/` - 页面组件
  - `router/` - Vue Router配置
  - `App.vue` - 根组件
  - `main.js` - 入口文件
- `public/` - 静态资源
- `index.html` - HTML模板
- `vite.config.js` - Vite配置
- `package.json` - 项目配置
