# 道义现实主义社会模拟仿真系统——项目研究介绍

---

## 一、项目概述

本项目是一个基于道义现实主义理论的**国际关系社会模拟仿真系统**。该系统采用**Agent-Based Modeling（ABM，基于智能体的建模）**方法，通过模拟国家领导人（智能体）在国际体系中的决策行为，研究国际秩序的演化规律。

### 1.1 研究目标

本研究旨在回答以下核心问题：

- **问题一**：不同类型的领导人如何影响国际秩序的形成与演变？
- **问题二**：在给定实力分布下，何种互动模式最有利于维护国际稳定？
- **问题三**：国际规范在不同领导类型主导下的有效性如何变化？
- **问题四**：随机事件如何改变国际体系的演化轨迹？

### 1.2 理论基础

#### 道义现实主义理论核心观点

道义现实主义是一种国际关系理论，其核心假设包括：

1. **国家利益优先**：国家的主要目标是维护和增强自身利益
2. **权力的双重性**：权力既是实现国家利益的手段，也可能被滥用
3. **道义的约束作用**：道义原则对国家行为具有约束力，但约束程度取决于领导人的类型
4. **秩序的建构性**：国际秩序是大国互动和规范建构的结果

#### 四种领导类型

| 领导类型 | 特征描述 | 决策倾向 | 行为边界 |
|---------|---------|---------|---------|
| **王道型** | 坚持道义，言行一致，优先保障体系公共利益 | 倾向于建立联盟、提供援助、签署条约、国际调停 | 禁止率先使用武力、单边制裁、单方面毁约 |
| **霸权型** | 选择性运用道义，坚持双重标准，以本国利益为核心 | 倾向于施加压力、维持联盟、遏制对手、提出国际规范 | 禁止单方面毁约 |
| **强权型** | 无视道义与国际规范，零和博弈思维，追求利益最大化 | 倾向于使用制裁、军事手段、经济施压 | 无特殊限制 |
| **昏庸型** | 无固定战略，决策高度个人化、反复无常 | 决策行为缺乏一致性 | 无限制 |

## 二、核心模型设计

### 2.1 国力计算模型：克莱因方程

本系统采用克莱因（Cline）方程来计算国家的综合实力：

```
P = (E + M) × (S + W)
```

其中：
- **P**：综合国力得分
- **E**：经济实力（GDP、贸易额、金融资源等）
- **M**：军事实力（军费、武器装备、军事人员等）
- **S**：战略目标（战略清晰度、战略定力等）
- **W**：国家意志（民族凝聚力、政府效能等）

### 2.2 实力层级划分模型

系统采用**正态分布方法**动态划分国家的实力层级，具体步骤如下：

1. 计算所有国家的克莱因P得分
2. 计算样本均值 μ 和标准差 σ
3. 对每个国家标准化为z分数：z = (P - μ) / σ
4. 根据z分数划分实力层级

| 实力层级 | z分数范围 | 理论比例 | 典型特征 |
|---------|----------|---------|---------|
| **超级大国** | z > 2.0 | ≈ 2.28% | 全球影响力，体系主导者 |
| **大国** | 1.5 < z ≤ 2.0 | ≈ 4.41% | 区域影响力，体系重要参与者 |
| **中等强国** | 0.5 < z ≤ 1.5 | ≈ 24.17% | 区域有限影响力，追随者或平衡者 |
| **小国** | z ≤ 0.5 | ≈ 69.15% | 影响力有限，主要受大国互动影响 |

**重要设计**：只有超级大国和大国需要配置领导人类型，中等强国和小国主要作为大国互动的响应者。

### 2.3 智能体模型

#### 智能体的构成

每个国家智能体包含以下核心属性：

- **基本属性**：国家ID、名称、所属区域
- **实力属性**：经济实力、军事实力、战略目标、国家意志、综合国力、实力层级
- **领导属性**：领导类型、核心偏好、行为边界、战略信誉度
- **学习属性**：决策历史、成功记录、偏好的演进
- **决策属性**：决策缓存、近期决策、决策模式

#### 智能体的决策流程

每个智能体在每轮仿真中经历以下决策阶段：

```
第一阶段：观察（Observation）
  ├── 观察其他国家的实力变化
  ├── 观察与当前各国的双边双边关系水平
  └── 观察环境中的活跃事件

第二阶段：分析（Analysis）
  ├── 识别当前面临的威胁和机遇
  ├── 评估与各国的合作潜力
  └── 分析自身资源状况和约束条件

第三阶段：决策（Decision）
  ├── 确定本轮本轮的战略目标
  ├── 评估所有可选的互动类型
  ├── 根据领导类型偏好选择
最佳互动
  └── 确定互动的目标和参数

第四阶段：执行（Execution）
  ├── 验证互动是否符合规则
  ├── 创建互动对象
  └── 执行互动并记录结果
```

#### 智能体的学习机制

智能体具备以下学习能力：

- **决策缓存**：对相似局势的决策进行缓存，避免重复计算
- **结果记录**：记录每次决策的结果，包括成功与否
- **偏好更新**：根据成功经验更新对互动类型的偏好
- **一致性检查**：检测新决策是否与近期决策冲突

## 三、互动系统

### 3.1 互动类型及其定义

系统共定义了**14种国际互动类型**，涵盖了从合作到对抗的完整谱系：

#### 合作型互动

| 互动类型 | 中文描述 | 典型效果 |
|---------|---------|---------|
| form_alliance | 建立联盟 | 大幅提升双边关系，消耗资源 |
| sign_treaty | 签署条约 | 提升双边关系，增加条约数量 |
| provide_aid | 提供援助 | 提升双边关系，消耗经济资源 |
| diplomatic_support | 外交支持 | 中等提升双边关系 |
| hold_summit | 举办峰会 | 根据结果提升或降低双边关系 |
| send_message | 发送信息 | 轻微提升或降低双边关系 |
| public_statement | 公开声明 | 轻微提升或降低双边关系 |
| cultural_influence | 文化影响 | 轻微提升双边关系 |

#### 对抗型互动

| 互动类型 | 中文描述 | 典型效果 |
|---------|---------|---------|
| declare_war | 宣战 | 极大降低双边关系，大量消耗资源，降低全球稳定度 |
| impose_sanctions | 实施制裁 | 大幅降低双边关系，消耗资源 |
| diplomatic_protest | 外交抗议 | 降低双边关系 |
| economic_pressure | 经济施压 | 中等降低双边关系 |
| military_posture | 军事部署 | 降低双边关系，可能引发安全担忧 |

### 3.2 互动规则与约束

系统对互动行为施加多种约束，确保仿真的真实性：

#### 关系约束

- **宣战约束**：只有在双边关系极为恶劣（<-0.5）时才能宣战
- **联盟约束**：只有在双边关系较为友好（>0.5）时才能建立联盟

#### 实力约束

- **宣战实力约束**：需要足够的综合实力支持（>100）
- **援助实力约束**：提供援助需要一定的经济实力（>50）

#### 频率约束

- 避免同一智能体在短时间内重复使用相同的互动类型（最近10轮内同一类型不超过5次）

#### 互斥约束

- 建立联盟与实施制裁互斥
- 提供援助与实施制裁互斥
- 签署条约与宣战互斥

### 3.3 互动影响计算详解

当互动被执行时，系统会精确计算其多层次影响。

#### 3.3.1 关系得分计算规则

双边关系得分是一个[-1.0, +1.0]区间的连续值：
- **+1.0** 表示最友好的关系（如最紧密的盟友）
- **0** 表示中立关系
- **-1.0** 表示最敌对的关系（如全面战争）

每种互动都会根据其**类型**和**强度参数**对双边关系产生特定影响：

**基础公式**：
```
新关系得分 = 旧关系得分 + 互动影响值 × 强度参数
```

**结果限制**：最终关系得分被限制在[-1.0, +1.0]区间内。

**详细影响表**：

| 互动类型 | 基础影响值 | 影响方向 | 适用条件 | 说明 |
|---------|-----------|---------|---------|-----|
| **建立联盟** | +0.3 | 改善 | 需关系>0.5 | 创建军事或政治联盟，大幅改善关系 |
| **签署条约** | +0.2 | 改善 | - | 签署国际条约或协定，改善关系 |
| **提供援助** | +0.25 | 改善 | 需实力>50 | 提供经济或技术援助，改善关系 |
| **外交支持** | +0.15 | 改善 | - | 提供外交支持，中等改善关系 |
| **首脑会晤(成功)** | +0.3 | 改善 | - | 成功峰会大幅改善关系 |
| **首脑会晤(失败)** | -0.2 | 恶化 | - | 失败峰会损害关系 |
| **发送友好信息** | +0.1 | 改善 | - | 轻微改善关系 |
| **发送敌对信息** | -0.1 | 恶化 | - | 轻微损害关系 |
| **公开赞扬** | +0.1 | 改善 | - | 轻微改善关系 |
| **公开批评** | -0.1 | 恶化 | - | 轻微损害关系 |
| **文化影响** | +0.1 | 改善 | - | 文化交流轻微改善关系 |
| **军事部署(防御)** | -0.1 | 恶化 | - | 防御性部署引发轻微软疑 |
| **军事部署(进攻)** | -0.3 | 恶化 | - | 进攻性部署大幅恶化关系 |
| **外交抗议** | -0.2 | 恶化 | - | 正式外交抗议损害关系 |
| **经济施压** | -0.3 | 恶化 | - | 经济施压中等恶化关系 |
| **实施制裁** | -0.5 | 恶化 | - | 经济制裁大幅恶化关系 |
| **宣战** | -1.0 | 恶化 | 需关系<-0.5，实力>100 | 战争使关系降至最低 |

**强度参数说明**：
- 大多数互动的强度参数默认为1.0
- 强度参数可调整互动的影响幅度
- 例如：强度为2.0的援助将产生0.5的关系改善（0.25 × 2.0）

**示例计算**：
```
初始情况：A国与B国关系为 0.2（轻微友好）

事件1：A国向B国提供援助（强度1.5）
影响 = +0.25 × 1.5 = +0.375
新关系 = 0.2 + 0.375 = 0.575

事件2：两国随后举行峰会（结果为成功）
影响 = +0.3
新关系 = 0.575 + 0.3 = 0.875（接近最友好关系）
```

#### 3.3.2 实力影响计算规则

互动会改变发起国的实力，主要体现为资源消耗：

| 互动类型 | 实力影响值 | 影响方向 | 影响因素 |
|---------|-----------|---------|---------|
| **提供援助** | 援助金额 × 0.1 × 强度 | 减少 | 援助金额、强度 |
| **实施制裁** | -5.0 × 强度 | 减少 | 强度 |
| **宣战** | -20.0 | 大幅减少 | 固定消耗 |
| **首脑会晤** | -1.0 | 轻微减少 | 固定消耗 |

**示例计算**：
```
A国提供100单位援助（强度1.5）
实力变化 = -100 × 0.1 × 1.5 = -15.0
```

#### 3.3.3 全局指标影响计算规则

某些互动会改变国际体系的整体状态：

| 互动类型 | 影响指标 | 影响值 | 说明 |
|---------|---------|--------|-----|
| **宣战** | 全球稳定度 | -0.1 | 战争降低全球稳定度 |
| **建立联盟** | 联盟数量 | +1.0 | 新增一个联盟 |
| **签署条约** | 条约数量 | +1.0 | 新增一个条约 |
| **提供援助** | 发展合作水平 | +0.1 | 促进全球发展合作 |

#### 3.3.4 第三方效应计算规则

某些重要互动会对未对直接参与的国家产生连锁影响：

**联盟建立后的第三方效应**：

```
对于每个第三方国家C：
  如果 C与A的关系 > 0.5 或 C与B的关系 > 0.5：
    C的安全担忧 += 0.1
    C对A的关系 -= 0.05
    C对B的关系 -= 0.05
```

逻辑：当两个友好国家建立联盟时，与其友好的第三方可能感到安全威胁。

**战争爆发后的第三方效应**：

```
对于每个第三方国家C：
  如果 C与目标国B的关系 > 0.5：
    C对A的关系 -= 0.2（对攻击者敌意增加）
    C的安全担忧 += 0.15
```

逻辑：当一国的盟友受到攻击时，该国会对攻击者产生强烈敌意。

#### 3.3.5 影响传播模型

系统提供三种事件影响传播模型：

1. **基于距离的传播（Distance-Based）**
   - 影响随地理距离衰减
   - 适用于领土争端、自然灾害等区域性事件

2. **基于关系的传播（Relationship-Based）**
   - 影响沿关系网络传播
   - 适用于联盟、制裁等政治性事件

3. **混合传播模型（Hybrid，默认）**
   - 结合距离和关系因素
   - 综合计算最符合现实的传播路径

**混合传播计算公式**：
```
影响强度 = 基础影响 × 关系权重 × 距离衰减因子

其中：
关系权重 = 0.3 + 0.7 × |双边关系|
距离衰减因子 = 1 / (1 + 距离 / 影响半径)
```

#### 3.3.6 决策优先级评估规则

智能体会根据当前局势评估决策的优先级，优先处理紧急情况：

| 优先级 | 判断条件 | 说明 |
|-------|---------|-----|
| **紧急** | 威胁等级 ≥ 0.9 | 需要立即处理，如面临生存威胁 |
| **高** | 威胁等级 ≥ 0.7 或 机遇等级 ≥ 0.8 | 重要威胁或重大机遇 |
| **中等** | 时间敏感度 ≥ 0.7 或 机遇等级 ≥ 0.5 | 需要及时处理或存在机遇 |
| **低** | 威胁等级 ≥ 0.4 | 存在中等威胁 |
| **常规** | 无特殊情况 | 日常事务处理 |

## 四、国际规范系统

### 4.1 国际规范的定义

系统预置了四项核心国际规范，每项规范具有以下属性：

| 规范ID | 规范名称 | 规范描述 | 初始有效性 | 初始遵守率 |
|-------|---------|---------|-----------|-----------|
| sovereign_equality | 主权平等原则 | 所有国家主权平等 | 100% | 100% |
| no_use_of_force | 不使用武力原则 | 禁止在国际关系中使用武力 | 100% | 100% |
| pacta_sunt_servanda | 条约必须信守原则 | 条约必须遵守 | 100% | 100% |
| non_intervention | 不干涉内政原则 | 不干涉他国内政 | 100% | 100% |

### 4.2 规范有效性计算

规范有效性随着国际事件的发生而动态变化：

```
新有效性 = 旧有效性 + 事件影响 × 领导类型调整因子
```

**事件影响示例**：

| 事件类型 | 对规范的影响 | 影响幅度 |
|---------|------------|---------|
| 宣战 | 不使用武力原则 | -15.0 |
| 签署条约 | 条约必须信守原则 | +5.0 |
| 违反条约 | 条约必须信守原则 | -10.0 |
| 干涉内政 | 不干涉内政原则 | -8.0 |
| 尊重主权 | 主权平等原则 | +3.0 |
| 侵犯主权 | 主权平等原则 | -12.0 |

**领导类型调整因子**：

| 领导类型 | 对道义规范的维护倾向 |
|---------|-------------------|
| 王道型 | +1.5（积极维护规范） |
| 霸权型 | +0.5（选择性维护规范） |
| 强权型 | -0.5（忽视规范） |
| 昏庸型 | 0.0（无明确态度） |

**示例计算**：
```
假设"不使用武力原则"当前有效性为80

事件：霸权型领导国宣战
事件影响 = -15.0
领导调整因子 = +0.5
新有效性 = 80 + (-15.0 × 0.5) = 72.5
```

### 4.3 规范遵守率计算

规范遵守率基于近期各国行为统计：

```
规范遵守率 = (遵守行为次数 / 相关行为总次数) × 100%
```

**计算窗口**：最近20轮内的所有相关行为。

### 4.4 规范对互动决策的约束

规范有效性会通过以下方式影响智能体的决策：

1. **高有效性规范（>70%）**：对决策产生较强约束
   - 例如：当"不使用武力原则"有效性高时，智能体更难选择宣战

2. **中等有效性规范（40%-70%）**：对决策产生中等约束
   - 智能体可能选择违反规范的互动，但需要额外的理由

3. **低有效性规范（<40%）**：对决策产生弱约束
   - 规范几乎不再具有约束力

## 五、事件系统

### 5.1 周期性事件

系统会定期生成周期性事件，模拟国际体系的自然节奏：

| 事件类型 | 触发周期 | 影响描述 |
||-------|---------|---------|
| 领导换届 | 每4年 | 领导层换届选举，可能改变政策方向 |
| 领导变更 | 选举后1轮 | 新领导上任，外交政策调整 |
| 季节更替 | 每1轮 | 春夏秋冬循环，影响经济活动 |
| 经济周期 | 每8年 | 复苏→繁荣→衰退→萧条循环 |
| 实力更新 | 每1轮 | 各国实力数据年度更新 |

### 5.2 随机事件

系统会随机生成外部事件，模拟国际体系的不确定性：

| 事件类型 | 基础概率 | 冷却期 | 影响描述 |
|---------|---------|--------|---------|
| 自然灾害 | 5% | 12轮 | 地震、洪水、干旱等，损害基础设施 |
| 经济危机 | 3% | 20轮 | 金融危机、货币贬值等，影响经济实力 |
| 技术突破 | 2% | 15轮 | 重大技术进步，增强国家实力 |
| 外交事件 | 8% | 8轮 | 外交照会、联合声明等 |
| 区域冲突 | 4% | 15轮 | 边境冲突、代理人战争 |
| 领土争端 | 6% | 10轮 | 边界争议、岛屿主权问题 |
| 盟友背叛 | 3% | 18轮 | 盟友退出联盟或倒戈 |
| 公共卫生危机 | 2% | 25轮 | 大流行病等全球性危机 |

### 5.3 事件概率的动态调整

事件的基础概率会根据以下因素动态调整：

**调整公式**：
```
实际触发概率 = 基础概率 × 实力层级因子 × 季节因子
```

**实力层级因子**：

| 事件类型 | 超级大国/大国 | 中等强国 | 小国 |
|---------|-------------|---------|-----|
| 自然灾害 | 0.8 | 1.0 | 1.2 |
| 经济危机 | 1.3 | 1.0 | 0.7 |
| 技术突破 | 1.5 | 1.0 | 0.5 |
| 外交事件 | 1.2 | 1.0 | 0.8 |

**季节因子**：

| 事件类型 | 春季 | 夏季 | 秋季 | 冬季 |
|---------|-----|-----|-----|-----|
| 自然灾害 | 1.0 | 0.8 | 1.2 | 1.5 |
| 经济危机 | 0.9 | 1.0 | 1.1 | 1.2 |
| 公共卫生危机 | 1.2 | 0.8 | 1.1 | 1.3 |

### 5.4 事件的优先级管理

系统采用优先队列管理事件，确保重要事件优先处理：

| 优先级 | 类型 | 说明 |
|-------|-----|-----|
| CRITICAL | 0 | 紧急事件（如战争、重大危机） |
| HIGH | 1 | 高优先级（如重大外交事件） |
| NORMAL | 2 | 正常优先级（如常规事件） |
| LOW | 3 | 低优先级（如通知一般事件） |

### 5.5 事件影响计算

事件会对国家实力和关系产生影响，影响程度由事件类型和影响级别决定：

| 事件类型 | 实力影响范围 | 关系影响 |
|---------|------------|---------|
| 自然灾害 | -0.6 至 -0.9 × 综合实力 | 对相关国家关系无直接影响 |
| 经济危机 | -0.5 至 -0.8 × 经济实力 | 可能引发其他国家援助或制裁 |
| 技术突破 | +0.4 至 +0.7 × 创新能力 | 提升国家国际影响力 |
| 公共卫生危机 | -0.5 至 -0.8 × 综合实力 | 可能触发国际合作 |

## 六、仿真流程

### 6.1 初始化阶段

1. **创建国家智能体**
   - 指定国家的基本属性（ID、名称、区域）
   - 设置初始实力指标（经济、军事、战略、意志）
   - 暂不分配领导类型（待实力层级确定后分配）

2. **计算实力层级**
   - 使用克莱因方程计算各国的综合国力
   - 使用正态分布方法划分实力层级
   - 确定哪些国家为超级大国、大国、中等强国或小国

3. **分配领导类型**
   - 为超级大国和大国分配领导类型（王道型、霸权型、强权型或昏庸型）
   - 设置各国的核心偏好和行为边界

4. **初始化环境**
   - 初始化国际规范（主权平等、不使用武力、条约必须信守、不干涉内政）
   - 初始化事件调度器
   - 设置初始季节为春季

### 6.2 仿真循环（多轮迭代）

每轮仿真包含以下步骤：

```
每轮开始：
  ├── 1. 更新轮次和季节
  ├── 2. 触发周期性事件（选举、季节更替、经济周期等）
  ├── 3. 生成随机事件（自然灾害、外交事件等，按概率）
  └── 4. 执行到期的延迟事件

智能体决策阶段：
  对于每个国家（按实力排序或随机顺序）：
      ├── 观察环境状态
      ├── 分析威胁与机遇
      ├── 选择互动类型和目标
      ├── 执行互动
      └── 记录互动结果

影响计算阶段：
  ├── 1. 计算所有互动的关系影响
  ├── 2. 计算所有互动的实力影响
  ├── 3. 计算所有互动的第三方效应
  ├── 4. 更新国际规范的有效性
  └── 5. 更新全局指标（稳定度、联盟数等）

数据收集阶段：
  ├── 1. 记录本轮所有互动
  ├── 2. 计算并记录统计指标
  ├── 3. 更新可视化数据集
  └── 4. 实时推送状态（如启用）

每轮结束：
  ├── 1. 检查终止条件
  ├── 2. 保存检查点（如到达间隔）
  └── 3. 活跃事件移入历史
```

### 6.3 仿真结束

1. **生成最终报告**
   - 汇总各轮的互动记录
   - 计算最终统计指标

2. **导出结果数据**
   - 支持JSON、Excel等格式
   - 包含互动历史、实力变化、关系变化等

3. **生成可视化图表**
   - 实力趋势图
   - 互动热力图
   - 关系网络图
   - 规范有效性曲线

## 九、系统架构

本系统采用**领域驱动设计（DDD）**架构，将系统划分为清晰的层次：

### 9.1 领域层（核心业务逻辑）

- **智能体领域**：国家智能体、大国智能体、小国智能体
- **互动领域**：互动规则、影响计算、第三方效应
- **事件领域**：事件生成、事件调度、事件影响
- **环境领域**：环境状态管理、国际规范维护

### 9.2 应用层（业务流程编排）

- **决策引擎**：协调智能体的决策过程
- **工作流管理**：单轮仿真、多轮仿真、检查点管理
- **数据分析**：统计指标计算、趋势分析

### 9.3 基础设施层（技术支撑）

- **AI引擎**：大语言模型调用
- **日志系统**：结构化日志记录
- **存储引擎**：数据持久化
- **性能监控**：性能指标收集

### 9.4 接口层（对外接口）

- **API接口**：RESTful API定义
- **WebSocket**：实时数据推送

道义现实主义社会模拟仿真系统是一个综合性的国际关系研究平台，具有以下特点：

1. **理论严谨**：基于道义现实主义理论，模型设计有清晰的学术基础
