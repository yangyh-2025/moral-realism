# 国内冲突预测脚本修改说明

## 修改完成时间
2026年4月2日

## 主要修改内容

### 1. 时间范围调整
- **原始数据范围**: 1961-2024年
- **预测目标**: 预测2025年，并递推预测2026年
- **使用数据**: 1962-2024年（实际数据中的年份范围）

### 2. 变量设置

#### 因变量
- `UCDP_Civil_conflict_0_1_2`（三分类，0/1/2）
  - 0: 无冲突
  - 1: 低强度冲突
  - 2: 高强度冲突

#### 自变量（共10项）

| 变量名 | 说明 | 取值范围 |
|--------|------|---------|
| polity | 政体得分 | -10到10（整数） |
| GDP_per_capita_ln | 人均GDP对数值 | 大于0 |
| GDP_growth | GDP增长率 | 无限制 |
| population_size | 0-14周岁人口占比 | 0到1 |
| trade | 贸易额占国民生产总值比例 | 大于0 |
| ethnic_diversity | 种族分化指数 | 0到1 |
| national_capacity | 国家综合国力指数 | 0到1 |
| government | 政府吸纳度 | 0/1/2/3（整数） |
| congress | 议会吸纳度 | 1到7（0.5步长） |
| ucdp_neighbor_confilctnumber | 发生冲突的邻国数量 | 0到正无穷（正整数） |

### 3. 随机森林参数设置

使用expanding-window三折时间序列交叉验证：

- **第一折**: 训练集1963-1974年（12年），验证集1975-1986年（12年）
- **第二折**: 训练集1963-1986年（24年），验证集1987-1998年（12年）
- **第三折**: 训练集1963-1998年（36年），验证集1999-2010年（12年）

- **评估指标**: 宏观F1得分（macro F1）

- **超参数搜索空间**:
  - SMOTE近邻数: [3, 5]
  - 决策树数量: [200]
  - 最大树深: [6, 10, None]
  - 叶节点最小样本数: [1, 3]
  - 类别权重: [None, "balanced_subsample"]

### 4. 非连续时间序列预测处理

对于非连续变量，使用以下方法进行预测：

#### 使用ARIMA模型预测的连续变量：
- `GDP_per_capita_ln`: 人均GDP对数值
- `GDP_growth`: GDP增长率
- `trade`: 贸易额占比

#### 使用简单外推方法预测的非连续变量：
- `polity`: 使用线性外推或移动平均，然后四舍五入到整数（-10到10）
- `government`: 使用线性外推或移动平均，然后四舍五入到整数（0到3）
- `congress`: 使用线性外推或移动平均，然后四舍五入到0.5倍数（1到7）
- `ucdp_neighbor_confilctnumber`: 使用最近几年的平均值，四舍五入为正整数
- `population_size`: 使用线性外推或移动平均，限制在0到1
- `ethnic_diversity`: 使用线性外推或移动平均，限制在0到1
- `national_capacity`: 使用线性外推或移动平均，限制在0到1

### 5. 分类阈值优化

- **不使用固定阈值**
- **使用随机森林的直接预测结果**（predict方法）
- 随机森林内部使用最优F1方法确定分类阈值

### 6. 注释说明

- 每个函数的功能说明
- 每个参数的作用和范围

## 脚本功能概述

### 完整的预测流程

1. **数据准备阶段**:
   - 加载Excel数据文件
   - 生成2025年的自变量预测
   - 构建滞后特征
   - 创建国家代码映射

2. **模型选择阶段**:
   - 使用expanding-window三折交叉验证
   - 网格搜索最优超参数
   - 使用宏观F1得分作为评估指标

3. **模型评估阶段**:
   - 在验证集（2011-2024年）上评估模型
   - 计算分类报告、混淆矩阵、ROC曲线
   - 输出各类别和总体指标

4. **递推回测阶段**:
   - 模拟真实预测流程
   - 预测下一年和下下一年
   - 将预测结果作为观测值
   - 计算预测准确率



5. **最终预测阶段**:
   - 使用所有数据拟合最终模型
   - 预测2025年冲突水平
   - 将2025年预测结果注入数据集
   - 递推预测2026年冲突水平

6. **结果保存阶段**:
   - 保存包含预测数据的数据集
   - 保存预测结果（2025年和2026年）
   - 保存回测结果
   - 保存评估表格和指标
   - 保存ROC曲线图
   - 保存模型对象

6. **全面评估**: 提供多种评估指标和可视化
