## 系统概述

### 什么是道义现实主义仿真系统？

本系统是一个基于道义现实主义理论框架的多智能体社会模拟仿真平台，用于：

- 研究国际关系中各国的互动模式
- 分析战略决策对国际秩序的影响
- 探索不同情境下国家行为的演变规律
- 验证国际关系理论假设

### 系统特点

- **LLM 驱动**: 使用大型语言模型模拟国家决策
- **智能体多样性**: 支持超级大国、大国、中等强国、小国等不同类型
- **实时仿真**: 支持 WebSocket 实时推送仿真状态
- **丰富可视化**: 提供多种图表展示仿真结果

## 核心概念

### 智能体 (Agent)

智能体代表参与国际关系的国家实体。每个智能体具有：

- **实力指标**: 基于 R.S. 克莱因的国力方程五要素
- **领导类型**: 描述国家领导人的决策风格
- **决策历史**: 记录所有决策和互动
- **关系网络**: 与其他智能体的关系状态

### 仿真轮次 (Rounds)

仿真按轮次进行，每轮代表一个时间周期（可配置）。每轮中：

1. 感知环境变化
2. 评估当前局势
3. 做出战略决策
4. 执行互动行为
5. 更新状态和关系
6. 记录观测数据

### 事件 (Events)

事件是影响系统状态的外部或内部因素，包括：

- **随机事件**: 自动生成的突发事件
- **用户事件**: 用户自定义的事件
- **周期性事件**: 按固定周期触发的事件

## 仿真管理

### 创建新仿真

通过前端界面：

1. 导航到"仿真管理"页面
2. 点击"创建仿真"按钮
3. 配置仿真参数：
   - 总轮次 (1-1000)
   - 每轮持续时间 (1-24月)
   - 随机事件概率 (0-1)
4. 选择保存路径
5. 点击"创建"

通过 API：

```bash
curl -X POST http://localhost:8000/api/simulation/create \
  -H "Content-Type: application/json" \
  -d '{
    "config": {
      "total_rounds": 100,
      "round_duration_months": 6,
      "random_event_probability": 0.1
    },
    "name": "我的仿真",
    "description": "测试仿真场景"
  }'
```

### 配置智能体

添加智能体到仿真：

**超级大国示例**:

```json
{
  "agent_id": "us",
  "name": "美国",
  "region": "北美",
  "agent_type": "superpower",
  "leader_type": "pragmatic",
  "power_metrics": {
    "critical_mass": 95,
    "economic_capability": 90,
    "military_capability": 92,
    "strategic_purpose": 88,
    "national_will": 85
  },
  "diplomatic_style": "assertive"
}
```

**小国示例**:

```json
{
  "agent_id": "small_nation",
  "name": "小国",
  "region": "亚洲",
  "agent_type": "small",
  "power_metrics": {
    "critical_mass": 30,
    "economic_capability": 25,
    "military_capability": 20,
    "strategic_purpose": 35,
    "national_will": 40
  },
  "survival_strategy": "bandwagoning"
}
```

### 启动仿真

1. 确保已配置至少 2 个智能体
2. 点击"启动仿真"按钮
3. 选择仿真模式：
   - **连续模式**: 自动运行到结束
   - **单步模式**: 每轮手动触发
4. 监控仿真进度

### 控制仿真

- **暂停**: 暂停当前仿真，保留状态
- **恢复**: 从暂停处继续
- **停止**: 终止仿真，保存当前结果
- **重置**: 重新开始仿真

## 智能体管理

### 查看智能体列表

访问"智能体管理"页面查看所有已配置的智能体。

### 修改智能体配置

1. 在智能体列表中点击"编辑"
2. 修改需要调整的参数
3. 保存更改

**注意**: 仿真开始后，部分参数将无法修改。

### 查看智能体状态

实时查看智能体的：

- 当前实力指标
- 决策历史
- 与其他智能体的关系
- 影响力趋势

## 事件管理

### 查看事件日志

在"事件管理"页面查看：

- 所有历史事件
- 事件影响范围
- 事件类型分类

### 创建自定义事件

```json
{
  "event_id": "custom_event_1",
  "name": "重大科技突破",
  "type": "technological",
  "description": "某国取得重大技术突破",
  "round": 50,
  "target_agents": ["target_agent_id"],
  "impact": {
    "power_change": 10,
    "relations_change": {
      "other_agent": 15
    }
  }
}
```

### 触发事件

手动触发某个事件：

1. 在事件列表中找到目标事件
2. 点击"触发"
3. 观察事件影响

## 数据分析与可视化

### 实时监控

仿真运行时可以实时查看：

- 仪表盘：核心指标概览
- 趋势图：各指标随时间变化
- 网络图：智能体关系网络
- 时间线：决策和事件时间轴

### 导出数据

支持多种导出格式：

- **JSON**: 完整仿真数据
- **CSV**: 表格化指标数据
- **Excel**: 多工作表数据分析

导出位置：

```bash
# 导出为 JSON
curl -X POST http://localhost:8000/api/export/json \
  -d '{"simulation_id": "sim_id", "path": "output/result.json"}'

# 导出为 CSV
curl -X POST http://localhost:8000/api/export/csv \
  -d '{"simulation_id": "sim_id", "path": "output/data.csv"}'
```

### 可视化组件说明

#### 实力趋势图

显示各智能体实力指标随轮次的变化趋势。

**交互功能**:
- 悬停查看详细数值
- 点击图例切换显示/隐藏
- 缩放和平移查看特定区间

#### 关系网络图

展示智能体之间的外交关系网络。

**节点属性**:
- 大小：代表实力强弱
- 颜色：代表所属区域
- 边：代表关系强度

**交互功能**:
- 拖拽调整布局
- 双击查看智能体详情
- 滚轮缩放视图

#### 决策时间线

按时间顺序显示智能体的决策和互动行为。

**显示内容**:
- 决策类型和内容
- 执行的互动行为
- 触发的事件
- 决策结果影响

## 最佳实践

### 仿真配置建议

1. **轮次设置**: 根据研究目的选择合适的轮次
   - 快速验证: 10-50 轮
   - 标准研究: 100-200 轮
   - 深度分析: 500+ 轮

2. **智能体数量**: 建议至少 4 个智能体以形成复杂互动
   - 最小配置: 2-3 个
   - 推荐配置: 4-8 个
   - 复杂场景: 10+ 个

3. **事件概率**: 根据研究需要调整
   - 稳定环境: 0.05-0.1
   - 动态环境: 0.15-0.25
   - 高不确定: 0.3+

### 数据管理

1. **定期备份**: 重要的仿真结果定期备份
2. **命名规范**: 使用有意义的仿真名称
3. **版本控制**: 重要配置使用版本控制

### 性能优化

1. **减少实时更新**: 复杂仿真降低更新频率
2. **批量导出**: 定期批量导出而非实时
3. **合理缓存**: 利用系统的缓存机制

## 故障排除

### 仿真卡住

1. 检查 API 密钥是否有效
2. 查看后端日志确认错误
3. 尝试重启仿真

### 智能体行为异常

1. 确认智能体配置正确
2. 检查领导类型设置
3. 查看决策历史了解问题

### 数据不一致

1. 检查数据库完整性
2. 验证配置参数
3. 重新初始化仿真
