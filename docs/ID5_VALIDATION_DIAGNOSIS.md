## 二、问题诊断

### 问题 A：追随行为 F1 = 0.43（Precision=0.31, Recall=0.69）

**核心矛盾**：Precision 极低（0.31），FP=487 远大于 TP=221。这意味着大量国家**不该追随却追随了**。

#### 根因 A1：中立国缺乏"保持中立"的强引导

**数据证据**（Round 1 实际追随关系）：
| 国家 | 历史应为 | 仿真实际 | 判定 |
|------|----------|----------|------|
| 西班牙(100) | 中立 | 追随德国 | ❌ FP |
| 瑞典(103) | 中立 | 追随德国 | ❌ FP |
| 比利时(101) | 中立 | 追随英国 | ❌ FP |
| 荷兰(104) | 中立 | 追随英国 | ❌ FP |
| 葡萄牙(106) | 中立 | 追随英国 | ❌ FP |
| 丹麦(107) | 中立 | 追随英国 | ❌ FP |
| 瑞士(108) | 中立 | 追随英国 | ❌ FP |

**代码根因**：`prompt_templates.py` 的 `FOLLOWER_VOTE_CONTEXT`（第848-863行）决策步骤中：
```
步骤1: 排除战争关系 → 步骤2: 警惕冲突关系 → 步骤3: 从盟友/伙伴/中立候选人中选择
```

问题在于步骤3——它**没有"保持中立"作为默认选项**。小国被要求从候选人中"选一个"，而非"默认中立，除非有充分理由跟随"。虽然步骤4提到了"若无合适候选人则中立"，但这被放在最后一步，暗示中立是"无奈之选"而非"理性默认"。

**per_agent_accuracy 数据佐证**：
- 中立国准确率极低：西班牙 0.04、比利时 0.04、瑞典 0.06、荷兰 0.10、瑞士 0.10
- 说明几乎所有中立国都被错误地引导去追随某国

#### 根因 A2：追随系统提示词中"中立偏好"缺失

`FOLLOWER_SYSTEM_PROMPT_TEMPLATE`（第760-804行）的通用决策规则中：
```
规则2: 追随≠同盟...评估每个候选人前先核对战略关系
规则4: 考虑历史互动模式
规则5: 考虑国力变化趋势
```

**缺少**：没有一条规则说"如果你是小国且与候选人无特殊战略利益关系，应默认选择中立"。所有规则都在讨论"如何选择候选人"，而非"是否应该选择候选人"。

#### 根因 A3：领导竞争参与决策过于宽松

`simulation_service.py` 第1210-1217行：
```python
big_powers = [
    agent for agent in agents
    if agent.get('power_level') in [
        PowerLevelEnum.SUPERPOWER.value,
        PowerLevelEnum.GREAT_POWER.value
    ]
]
```

所有超级大国和大国自动参与领导竞争。在19国体系中，CINC前5%≈前1名为超级大国，前15%≈前3名为大国。这意味着德国(0.237)、俄国(0.189)、英国(0.187)都自动成为候选人。

**问题**：大国自动参选 → 总有候选人 → 小国总能看到候选人 → 总有"可追随对象"。历史上许多小国（西班牙、瑞典、瑞士、荷兰、丹麦）在整个一战期间保持中立，但仿真中因为"总有候选人可选"而被迫选择追随。

#### 根因 A4：验证脚本中 SUPERPOWERS 设定与仿真不一致

`validate_id5_f1.py` 第51行：
```python
SUPERPOWERS = {94}  # Only UK is superpower
```

但从 CINC 值看，德国(0.237) > 俄国(0.189) > 英国(0.187)，19国体系中前5%≈前1名，**德国应该是超级大国**，而非英国。

这意味着验证脚本将英国排除在追随评估之外（作为superpower），但实际上德国（而非英国）才是仿真中被分类为超级大国的国家。这导致：
- 如果德国在仿真中是superpower → 德国自动中立 → 评估正确
- 如果德国不是superpower → 德国本应参与追随但被错误排除 → 评估偏差

### 问题 B：行为分布 Macro F1 = 0.25

#### 根因 B1：领导类型权重严重偏向军事行为

`prompt_templates.py` 第220-229行，强权型（俄国的领导类型）：
```python
"1. 军事/强制类行为：额外+0.2战略威慑价值"
"2. 对弱小对手使用武力：额外+0.1收益"
"3. 尊重主权的行为：无额外道义收益"
"4. 合作类行为：视为工具性手段，无道义加成"
"5. 非尊重主权行为：无声誉损失惩罚"
```

而霸权型（德国的领导类型）第216-218行：
```python
"4. 军事威慑类行为：+0.1战略价值"
"5. 对弱小对手使用对抗行为：成本降低0.1"
```

**问题**：强权型对军事行为有 +0.2 加成，但对经济/合作行为没有任何加成。霸权型也有 +0.1 军事加成。这导致 LLM 在成本收益分析中系统性地高估军事行为的收益。

#### 根因 B2：经济手段 power_change 为负值，LLM 规避

`action_manager.py` 经济手段的 power_change：
- 开展实质性合作(6)：发起方 +0.1，目标方 +0.1 → **合理**
- 提供援助(7)：发起方 **-0.03**，目标方 +0.15 → **发起方亏本！**

LLM 在做成本收益分析时，看到"提供援助"会让自己的国力下降，自然不选。但历史事实上，一战前各大国大量对外援助（如法国对俄国的贷款、英国对小国的经济支持）来巩固联盟。

#### 根因 B3：信息手段（威胁、调查）在 prompt 中被忽视

信息手段只有2项：调查(9)和威胁(13)。在 prompt 中没有针对信息手段的特殊引导。威胁(13)虽然 power_change 不错（发起方 +0.1，目标方 -0.2），但它被归类为"信息手段"，而非"军事手段"，LLM 可能在分类时混淆。

#### 根因 B4：行为分布 F1 计算方式只看5个强国

`validate_id5_f1.py` 第371行：
```python
major_countries = list(EXPECTED_ACTION_DOMINANT.keys())
```

只评估了5个强国（德国、俄国、英国、法国、奥匈帝国）。但奥匈帝国的准确率只有 9%，法国只有 35%，严重拉低了 Macro F1。

**数据佐证**（per_country accuracy）：
| 国家 | 期望主要行为 | 准确率 |
|------|------------|--------|
| 德国 | Military | 83% ✅ |
| 俄国 | Military | 97% ✅ |
| 英国 | Diplomatic | 60% ⚠️ |
| 法国 | Diplomatic | 35% ❌ |
| 奥匈帝国 | Military | 9% ❌ |

奥匈帝国只有9%的轮次以军事行为为主，但历史上奥匈帝国在一战前是非常军事化的（巴尔干危机、对塞尔维亚的最后通牒等）。问题可能是：奥匈帝国被分类为中等强国，其 prompt 中的领导类型引导不够强，导致 LLM 选择了大量外交行为而非军事行为。

## 三、改进方案

### 改进 1：追随决策——强化中立默认值

**修改文件**：`app/core/prompt_templates.py`

#### 1a. 在 `FOLLOWER_SYSTEM_PROMPT_TEMPLATE` 中添加"中立偏好"规则

在【通用决策规则】中添加：
```
7. 【中立偏好原则-适用于所有国家】在追随决策中，"中立"（不追随任何国家）是理性且可敬的选择。以下情况应优先选择中立：
   - 你的国家利益不直接受候选领导者影响
   - 你与所有候选者均无盟友/伙伴关系
   - 追随某一会让你卷入不必要的冲突风险
   - 你是小型中立国，追随任何一方都会损害你的独立性
   小国默认应选择中立，除非有明确的战略利益理由追随某一方。
```

#### 1b. 修改 `FOLLOWER_VOTE_CONTEXT` 的决策步骤

将步骤3从"从候选人中选择"改为"追随必要性评估"（**关键：追随≠同盟**）：
```
【决策步骤-请严格按顺序执行】
1. 先排除：标注 ⚠️【战争关系-禁止追随】的，原则上严禁选择
2. 再警惕：标注 ⚠️【冲突关系-应避免追随】的，仅在有明确战略利益时才考虑
3. 【关键步骤-追随必要性评估】在考虑追随任何人之前，先问自己：我有必要在本轮议题上追随某国吗？
   - 追随≠同盟：追随是在特定议题上对某国领导偏好的表态，不是战略结盟
   - 但追随仍需理由：追随意味着让渡部分决策自主权，需要有实际收益
   - 以下情况应优先选择中立：
     a. 长期奉行中立/独立外交的国家（瑞士、瑞典、西班牙、荷兰、丹麦、挪威等）
     b. 追随会让你卷入与你无关的大国冲突
     c. 所有候选者在当前议题上都没有为你提供明确的具体利益交换
   - 以下情况可以考虑追随：
     a. 某候选者在当前议题上为你提供了实质性利益
     b. 追随能帮助你获得独自无法获得的利益
     c. 你与某候选者在当前议题上有共同对手或共同利益
4. 若确实有必要追随：从候选人中，按实力、领导类型、当前议题利益、双向互动统计选择最优
5. 最终默认：中立（follower_agent_id = null）
```

#### 1c. 添加历史中立国提示

在追随投票的 user prompt 中，为一战场景添加：
```
【历史背景参考-1913年欧洲】
在当前历史语境下，以下国家通常保持外交中立，不主动追随大国阵营：
西班牙、比利时、瑞典、荷兰、葡萄牙、丹麦、瑞士、挪威
这些国家在大战爆发前通常奉行中立政策，追随大国阵营需要极强的战略理由。
```

17. 【行为多样性约束】为维护国家综合利益，你的行为组合应覆盖多种手段类别：

### 改进 3：追随决策——调整领导竞争参与机制

**修改文件**：`app/services/simulation_service.py`

#### 3a. 不自动让所有大国参选

将 `_generate_follower_decisions` 中的自动参选改为 LLM 决策参选（已有此逻辑但可能执行不够严格）。

#### 3b. 参与决策中添加"不应参与"的强引导

在 `LEADERSHIP_PARTICIPATION_CONTEXT` 中添加：
```
【不参与的适用情况】
- 你虽然有一定国力，但当前国际环境下不宜高调参与领导竞争
- 参与领导竞争可能引发其他大国的警惕和对抗
- 你的国家利益更适合作为独立行动者而非体系领导者
- 历史上你的国家在一战前时期通常不参与全球领导权争夺
```

### 改进 4：验证脚本修正

**修改文件**：`docs/validate_id5_f1.py`

#### 4a. 修正 SUPERPOWERS 定义

```python
# 原始：SUPERPOWERS = {94}  # Only UK is superpower
# 修正：基于仿真实际CINC值，德国(CINC最高)是超级大国
# 但UK在仿真中被追随最多(leader_agent_id=94)，所以UK在仿真中实际起到了领导作用
# 需要与仿真中的实际power_level保持一致

# 或者：不排除任何国家，让所有国家参与评估
```

#### 4b. 调整历史基准数据

考虑将中立国的评估从"精确匹配"改为"宽松匹配"——中立国只要不追随**错误阵营**的领导者就算正确（如西班牙不追随英国或德国均可）。
