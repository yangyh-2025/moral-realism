# PART 1: 塞尔维亚1914年决策研究

## 1.1 塞尔维亚对奥匈最后通牒的回复 — 接受了什么，拒绝了什么

塞尔维亚的回复被广泛认为是"非凡的让步"，几乎接受了奥匈帝国所有10项要求。具体而言：

**塞尔维亚接受的内容：**
- 在官方公报头版发表正式声明，谴责反奥宣传
- 将军队命令传达给塞尔维亚军队
- 解散反奥社会团体（包括"黑手党"相关组织 Narodna Obrana）
- 修改新闻法以压制反奥宣传
- 从军队、海军和文职部门中撤换任何可证明参与反奥宣传的人员
- 逮捕涉嫌参与萨拉热窝阴谋的人员

**塞尔维亚拒绝/保留的唯一要点（第6条）：**
奥匈帝国要求其官员**在塞尔维亚领土上**参与对萨拉热窝暗杀事件的调查和起诉。塞尔维亚拒绝，理由是这将"违反塞尔维亚宪法和刑事法律"，并构成对其**主权**的侵犯。塞尔维亚并未完全拒绝，而是表示只接受"符合国际法和睦邻关系"的参与方式，并提议将争议提交海牙国际法庭仲裁。

**关键引语：**
奥地利外交部起草最后通牒的参赞**亚历山大·穆苏林**（Alexander Musulin）称塞尔维亚的回复为"我所知道的最出色的外交技巧范例"。

**奥地利大使吉斯尔**（Wladimir von Giesl）被指示"不管塞尔维亚如何回复"都要断绝外交关系——战争的决策先于回复。

## 1.2 塞尔维亚的让步是真诚的还是策略性的？

学术界在此问题上存在分歧：
- **克拉克（Clark）和麦克米兰（MacMillan）：** 塞尔维亚知道奥地利无论如何都会发动战争，因此让步是一种旨在赢得国际舆论的外交策略——尤其在争取英国支持方面。
- **塞尔维亚反修正主义学者：** 让步是真诚的；塞尔维亚刚经历两次巴尔干战争，军队疲惫不堪，迫切需要避免另一场冲突。帕希奇首相确实希望和平。
- **现实评估：** 即使塞尔维亚在形式上让步，奥地利（正确地）不相信塞尔维亚会真正执行这些要求——塞尔维亚国家机构的渗透性和黑手党的权力使得任何关于镇压反奥活动的承诺都不可信。这是经典的"承诺可信度"问题。

## 1.3 塞尔维亚的主权观与"大塞尔维亚"计划

塞尔维亚的主权构想是其民族统一计划的核心——"大塞尔维亚"（或后来的"南斯拉夫"理念）本质上是一个**收复失地**（irredentist）计划：

**历史脉络：**
- 1844年 **《纳切尔塔尼耶》（Načertanije），伊利亚·加拉沙宁**（Ilija Garašanin）：塞尔维亚民族统一的大纲领，设想将波斯尼亚、黑塞哥维那、黑山、马其顿和部分阿尔巴尼亚领土纳入塞尔维亚版图。
- 塞尔维亚的建国神话将其定位为"巴尔干的皮埃蒙特"——相当于意大利统一运动中撒丁王国的角色，作为南斯拉夫民族统一的核心。
- **埃丁·哈伊达尔帕希奇**（Edin Hajdarpasic, *Whose Bosnia?*, 2015）：波斯尼亚在塞尔维亚和克罗地亚民族主义叙事中都是被竞相宣称的领土，早在1914年之前就已如此。

**与奥匈帝国的零和冲突：**
塞尔维亚的建国计划与哈布斯堡帝国的存在在根本上是不兼容的。正如一份关于*通过战争建国*的学术论文所述："奥地利-匈牙利领导人认定塞尔维亚的建国努力是对其帝国的致命威胁……越来越多的南部斯拉夫人——不仅是塞尔维亚人，还有克罗地亚人和斯洛文尼亚人——开始认同塞尔维亚主导的南斯拉夫理念，其实现必然以奥匈帝国的损失为代价。"

**德拉甘·巴基奇**（Dragan Bakić, *Balcanica* 2019/2020）指出，塞尔维亚战时政策中存在"大塞尔维亚"（兼并塞族聚居区）和"南斯拉夫"（塞族、克族、斯洛文尼亚族统一）之间的张力。帕希奇首相坚持南斯拉夫路线，但其部分外交官（如马特亚·博什科维奇）倾向于大塞尔维亚方案。

## 1.4 黑手党的角色 — 塞尔维亚政府与萨拉热窝暗杀

这是整个1914年危机中最具争议的问题：

**阿皮斯**（Dragutin Dimitrijević "Apis"）是塞尔维亚军事情报部门负责人，也是黑手党（"统一或死亡"）的创始人和实际领袖。
- 他曾领导1903年推翻奥布雷诺维奇王朝的政变（国王和王后被枪杀、刺戳和肢解）
- 1914年，他组织了萨拉热窝暗杀行动的招募、训练和跨境输送
- 三名年轻的波斯尼亚塞族刺客（普林西普、查布里诺维奇、格拉贝日）在塞尔维亚境内接受炸弹和枪法训练，通过黑手党的地下网络被偷运回波斯尼亚

**政府知情程度：**
- 首相帕希奇**确实被告知**有武装人员正在越境，但他是否知道具体的暗杀阴谋尚存争议
- 帕希奇下令逮捕越境者，但**命令未被边境卫兵执行**
- 黑手党执行委员会在得知阿皮斯的计划后**曾要求取消行动**，阿皮斯承诺发出反指令——但这些指令要么来得太晚，要么敷衍了事
- 历史学家**迈尔·别拉亚茨**（Mile Bjelajac）认为阿皮斯试图撤销行动

**克拉克（Clark）的立场：** 帕希奇听到了传言，私下谴责，但无法或不愿阻止——这是一个"国家容忍的国家恐怖主义"问题。

**扎梅蒂察（Zametica）的反驳：** 阿皮斯在1914年5-6月忙于策划在贝尔格莱德的政变以推翻文官政府，而非暗杀费迪南；阿皮斯本人是那位著名的"塞尔维亚警告维也纳"的来源，旨在**阻止**暗杀。

## 1.5 巴尔干战争（1912-1913）— 塞尔维亚的信心与扩张

巴尔干战争是塞尔维亚国家发展的关键转折点：
- **领土和人口翻倍：** 塞尔维亚获得了桑扎克、科索沃和马其顿的大片领土，领土和人口均翻倍
- **军事信心飙升：** 塞尔维亚军队积累了宝贵的实战经验，相继击败奥斯曼帝国和保加利亚
- **"第二个病夫"心态：** 随着奥斯曼帝国在欧洲的势力崩溃，塞尔维亚民族主义者开始将奥匈帝国视为"第二个病夫"，相信同样的解体情景可能发生在哈布斯堡帝国
- **受挫的亚得里亚海通道：** 奥地利成功游说创建独立阿尔巴尼亚以封锁塞尔维亚的亚得里亚海出海口——这使塞尔维亚深感愤怒，认为自己的战果被"偷走"
- **奥匈的软弱回应：** 维也纳在巴尔干战争期间调动军队四次，两次威胁塞尔维亚，但每次都退缩。这强化了塞尔维亚的看法：奥匈帝国是虚张声势

**与俄国的关系：** 俄国支持巴尔干同盟作为对抗奥地利的平衡力量，但巴尔干国家的野心很快超出了莫斯科的意图。俄国驻贝尔格莱德大使**哈特维希**（Nikolai Hartwig）的影响力极大，塞尔维亚被有效视为"巴尔干地区唯一完全服从俄国政治影响的国家"。

然而，塞尔维亚**不是一个简单的俄国代理人**。帕希奇有独立的议程；黑手党经常不受贝尔格莱德文官政府控制地行事。塞尔维亚有着主动利用大国对立来实现自身扩张目标的长期记录。

## 1.6 塞尔维亚对奥地利主权的看法

塞尔维亚不承认奥匈帝国对波斯尼亚-黑塞哥维那的主权主张（1908年正式吞并，此前自1878年起占领）。塞尔维亚民族主义将波斯尼亚视为"塞尔维亚土地"——哈布斯堡统治在那里被视为非法占领。

克拉克在《梦游者》中详细记录了塞尔维亚在波斯尼亚开展的反奥宣传、情报渗透和准军事网络建设。这些活动并不隐晦；它们是塞尔维亚民族计划的公开组成部分。

## 1.7 塞尔维亚是受害者、挑衅者，还是两者皆有？

学术界的立场分为清晰的三方：

| 阵营 | 代表学者 | 核心论点 |
|------|---------|---------|
| **塞尔维亚是受害者** | 扎梅蒂察（Zametica）、别拉亚茨（Bjelajac） | 奥匈帝国是侵略者；普林西普的意识形态来自哈布斯堡克罗地亚，而非塞尔维亚；黑手党在1914年5-6月忙于在贝尔格莱德策划政变，而非暗杀 |
| **塞尔维亚是挑衅者** | 克拉克（Clark）、麦克米兰（MacMillan）、麦克米金（McMeekin） | 塞尔维亚的收复失地计划是对奥匈帝国的持续侵犯；国家容忍/支持黑手党；塞尔维亚的暴力政治文化是危机的根本原因之一 |
| **两者皆有** | 大多数学术共识 | 塞尔维亚是国家支持的收复失地主义 + 合法民族自决的混合体；奥地利是侵略性的帝国 + 对恐怖主义作出反应的国家。两者都有部分正确，也都有过错 |

**麦克米兰（MacMillan）** 将塞尔维亚非正式网络比作伊朗支持真主党的模式——这一比较在塞尔维亚学术界引发强烈反弹。

**克拉克（Clark）** 的结构性独创在于他提出的 **"巴尔干引爆场景"**——法俄同盟已"将三个世界大国的国防政策与欧洲最暴力、最不稳定的地区的不确定命运捆绑在一起"。在这个框架下，塞尔维亚不是被动的受害者，而是系统中的积极破坏因素。

## 2.2 当前国家/智能体建模

**AgentBase 属性：**
- 6项CINC底层指标（milex, milper, irst, pec, tpop, upop）— 仿真全程固定
- CINC综合国力指数（initial + 每轮更新的current，0-1比例值）
- 实力层级（超级大国/大国/中等强国/小国）— 由极性-权力占比判定方案每轮动态计算
- 领导集体类型（王道型/霸权型/强权型/昏庸型）— 仅大国/超级大国可配置
- 国家利益偏好（由实力层级决定，与领导类型无关）

**核心建模特征：**
- 基于阎学通的道义现实主义框架的4种领导类型
- CINC大国权力测量体系
- 多极/两极/单极格局的动态极性判定
- 战略关系五级体系（战争/冲突/无外交/伙伴/盟友）

## 2.3 决策引擎分析

**已有机制：**
1. **20种GDELT标准互动行为：** 分为四大类（外交/经济/军事/信息），每种行为携带 respect_sov（是否尊重主权）、power_change（国力变化量）
2. **LLM驱动的决策：** 成本收益分析，领导类型行为权重，军事平衡评估（基于CINC比值）
3. **三层验证机制：** 行为ID校验、目标国校验、行为前置条件校验
4. **联盟链式卷入机制（chain-ganging）：** 明确参照"七月危机"建模，盟友被攻击→义务反应
5. **追随机制：** LLM驱动的追随投票，大国独立性保障，互相追随修复
6. **秩序判定：** 基于主权尊重率和追随分布，四种秩序类型

**提示词中的关键约束：**
- **规则13（同盟链式卷入）：** "此机制是「七月危机」式连锁反应的核心"
- **规则8（相对实力意识）：** 预防性威慑逻辑
- **规则11（冲突升级惯性）：** 路径依赖性
- **规则15（时间尺度约束）：** 冲突升级需要时间
- **强权型领导** 具有"激情干预"机制（个人声望、对臣服的敏感、军事压力下向军方让渡、决策波动性）

## 2.4 每轮仿真流程

```
每轮 = 3 个月（季度）
1. 创建轮次记录
2. 主动决策阶段（每个国家LLM决策，并发）
3. 响应决策阶段（每个国家LLM决策，并发）
4. CINC国力更新（基于行为记录全局重算）
5. 追随投票阶段（每个国家LLM决策）
6. 秩序判定
7. 战略关系演变
8. 战略目标评估（每10轮）
```

## 2.5 现有模型中缺失/薄弱的维度

基于对历史决策的学术分析，当前模型在以下方面存在空白：

| 维度 | 当前覆盖 | 缺口分析 |
|------|---------|---------|
| **王朝利益 vs 国家利益** | 仅"领导类型"编码了部分偏好差异 | 没有区分"王朝声望"与"国家安全"的机制——对奥地利决策至关重要 |
| **最后通牒/外交讹诈机制** | 存在"威胁"和"胁迫"行为，但它们只是通用行为 | 没有"设计为不可接受的外交文本"的概念——缺乏对"表态性外交"vs"真诚外交"的区分 |
| **国家支持的收复失地主义/恐怖主义** | 行为集中有"开展调查"和"抗议"，但没有间接侵略工具 | 没有代理战争、非国家行为者（如黑手党）、越境颠覆活动 |
| **国内政治—外交政策的联系** | 领导类型包含某些偏好，但没有制度化联系 | 没有议会/内阁限制、军方自主权、蒂萨式的否决权、军方让渡机制 |
| **"荣耀/声望"作为独立激励因素** | 仅在强权型领导的"激情干预"中提及 | 所有大国都可能为声望而行动——不是人格特质，而是结构性特征——目前尚未普遍建模 |
| **国际法/主权规范** | respect_sov 是一个二元标签 | 没有编纂为规范演变的梯度性——没有对"文明标准"、主权合法性认知的历史性差异进行建模 |
| **承诺可信度问题** | 行为系统假定执行 | 即使形式上让步，也没有判别对方是否会实际执行的能力——这是奥地利不信任塞尔维亚的核心原因 |
| **预防性战争逻辑** | 规则中隐含提及"相对实力意识" | 没有明确建模"现在打更好，而不是以后"的计算——这对1914年至关重要 |
| **大国—附庸国代理关系** | 追随机制存在，但更偏向自愿 | 没有不平等同盟动态——塞尔维亚-俄国的关系更像是"附庸国约束大国"，而非单纯追随 |
| **最后通牒/时间压力** | 不存在 | 当前只能选择行为列表中的标准行为——没有胁迫性外交/最后通牒机制 |
| **多民族帝国的内部解体压力** | 不存在 | CINC指标固定——国家是一个无内部差异的统一体，而在历史现实中，民族离心力是奥地利行为的核心驱动 |

# PART 3: 国际关系理论与道德哲学框架

## 3.1 现实主义 vs 建构主义：理解1914年的两种路径

**结构现实主义（沃尔兹）：**
- 战争归因于多极体系的"安全困境"：每个大国感到必须支持关键盟友（锁链式卷入），以避免被孤立和统治
- **戴尔·科普兰**（Dale Copeland）的"预防性战争"论：德国精英恐惧相对于正在崛起的俄国的不可逆转衰退——"窗口正在关闭"。一场局部化的奥塞战争是最佳选择，但大陆战争是可接受的风险
- **塞缪尔·威廉姆森**（Samuel R. Williamson）认为奥匈决策是"防御性现实主义"的范例："哈布斯堡君主国发动战争是为了惩罚塞尔维亚、实现政权更迭，并提醒世界它是一个大国……是出于生存考虑，而非扩张需要"

**建构主义：**
- 克拉克的论点隐含了建构主义逻辑：塞尔维亚与奥匈帝国之间的对抗"由欧洲帝国与民族之间的更广泛冲突所框定和加剧"
- 其他大国不再承认奥匈帝国安全利益的合法性；一个多民族帝国在日益由民族主义原则定义的欧洲中在规范上已经过时
- **建构主义的核心洞见：** 物质力量的变化无法完全解释危机——是**合法性的丧失**使奥地利-匈牙利在存在上感到不安全

**博弈论视角（弗兰克·扎加尔）：**
- 奥地利领导人**预期**塞尔维亚会为其主权而战——这使得塞尔维亚的反威胁"完全可信但完全无效，因为奥地利人已经将其纳入了考量"
- 奥地利的最后通牒不是一场"讨价还价"；它是发动"入侵的借口"

## 3.2 1914年的主权概念

**1914年国际法下的"主权"含义：**
- 在威斯特伐利亚传统中，主权意味着：（1）领土内的最高管辖权；（2）对外独立。但实践中，大国享有完全主权，而小国和"非文明"政体的主权是条件性的
- **"文明标准"（Standard of Civilization）：** 19世纪国际法将世界分为"文明"国家、"野蛮"国家和"未开化"国家。只有"文明"国家才享有完全主权。土耳其直到1856年才被接纳为"欧洲协调"成员。巴尔干国家长期被视为处于"文明"边缘
- 奥匈最后一个通牒的要点6——要求奥地利官员在塞尔维亚领土上进行调查——正是针对主权的核心要素：领土管辖权。这正是塞尔维亚拒绝的原因；也是奥地利故意放置的条款

**民族自决：**
- 在1914年，民族自决**不是**国际法中的普遍原则。它是19世纪民族主义的诉求（意大利统一、德国统一），但被帝国秩序所否定
- 这一原则仅在威尔逊的《十四点》（1918年）之后才在国际法中获得稳固地位
- 在1914年，塞尔维亚的民族统一主张是一种**政治诉求**，而非法律权利。奥地利将其视为颠覆活动，而非合法愿望

## 3.3 将七月危机应用于理论框架

**安全困境（Security Dilemma）：**
- 奥地利感到塞尔维亚收复失地主义对其生存的威胁——它在客观上是防御性的
- 塞尔维亚将其民族统一计划视为防御性的、合法的
- 俄国将奥地利的预防性打击视为进攻性的
- 德国将俄国动员视为进攻性的
- **结构现实主义的经典安全困境螺旋**——每一方都将自己的行动视为防御，而将对方的行动视为进攻

**进攻性现实主义 vs 防御性现实主义：**
- 米尔斯海默的进攻性现实主义预言：大国追求最大化权力，战争在寻求霸权时不可避免
- 防御性现实主义（威廉姆森）：奥地利的行动是为确保生存，而非扩张
- 哪一个更好？取决于你分别看待哪个阶段：**宣布的目标**是防御性的（确保帝国生存）；**实际的政策路径**越来越具有进攻性（战争期间推进至公开的领土扩张）

**博弈论（扎加尔的完美威慑理论）：**
- 奥塞危机是"完美威慑理论"的经典失败：奥地利已下定决心开战，因此任何塞尔维亚威胁都不可威慑
- 关键机制不是"威慑失败"，而是"选择效应"——奥地利**已选择**开战，使得外交信号变得无关紧要

## 3.4 正义战争理论（Jus ad Bellum）应用于奥地利决策

根据经典正义战争标准评估奥地利1914年的决策：

| 标准 | 奥地利的案例 | 评估 |
|------|------------|------|
| **正当原因（Just Cause）** | 报复暗杀；防止国家支持的恐怖主义 | 薄弱的理由——塞尔维亚作出了实质性让步，证据收集是事后的 |
| **正当意图（Right Intention）** | 官方：防御性，惩罚性 | 实际上：最后通牒设计为不可接受——展示的意图是侵略，而非自卫 |
| **最后手段（Last Resort）** | 外交努力？最后通牒后仅48小时；拒绝调解 | 明显地不满足——英国调解提议被德国/奥地利蓄意延迟 |
| **正当权威（Legitimate Authority）** | 在王朝/帝国内有正当权威 | 在形式上是满足的——但对主权的冲突性主张使这一点变得复杂 |
| **合理的成功前景** | 对塞尔维亚：高。对欧洲战争：极低 | 严重误判——他们知道俄国会介入 |
| **相称性（Proportionality）** | 欧洲战争超过任何正当目标 | 灾难性地不成比例；他们预期的更广泛的战争"不能仅仅因为报复暗杀而正当" |
| **公开宣战** | 形式上履行 | 满足——但形式掩盖了实质 |

**综合评估：** 根据just war标准，奥地利的决策近乎全面失败。唯一勉强满足的标准（正式宣战、皇室权威）是形式上的；实质标准（正当原因、最后手段、相称性）均未满足。

## 3.5 国际关系中的"荣耀"概念

- **现实主义传统：** "荣耀"被视为国家行为的非理性因素——对声望和地位的追求偏离了物质利益的最优化
- **建构主义批评：** 声望在战略上可能是理性的——它是"社会资本"，能震慑并吸引盟友。奥地利对自身"作为大国的声誉"的计算并非与安全计算分离；它**本身就是**安全计算的一部分
- **地位不一致性理论：** 国家在地位（被承认的排名）低于其物质能力（实际能力的排名）时会感到强烈的不安全感——这正是1914年奥匈帝国的情况：物质上处于大国之列，但越来越不被当作为大国对待
- **ABM实施：** 可以在现有领导类型权重的基础上，增加"声望"作为与物质利益平行的独立目标函数——在特定条件下，主体可能为维护声望牺牲物质利益

## 3.6 国际法中的侵略性战争

- **1914年的法律状况：** 没有公约或条约明确禁止侵略性战争。海牙公约（1899年、1907年）规范了战争的**实施**（jus in bello），但未规范诉诸战争本身（jus ad bellum）
- **凯洛格-白里安公约**（1928年）才宣布侵略性战争非法
- 在1914年之前，战争是神圣的国家特权——"诉诸战争的权利"（ius ad bellum）是主权的核心属性
- 这意味着奥地利发动战争的决策在**法律上**并非"非法"——但在**政治和道德上**仍受到"文明行为"规范的约束
- 克拉克等人强调：关键问题不是"合法性"，而是"合法性"——其他大国是否认为奥地利的行动正当
