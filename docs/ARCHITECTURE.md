# 系统架构文档

## 概述

道义现实主义社会模拟仿真系统采用四层架构设计，支持前后端分离部署。

## 架构图

```
[用户浏览器]
      |
      v
[前端 React应用] <---> [WebSocket] <---> [后端 FastAPI]
      |                                   |
      v                                   v
[RESTful API] <---> [核心引擎层]
                                      |
                                      v
                                 [数据存储 SQLite]
```

## 架构层次

### 1. 底层引擎层

负责核心功能实现：

- **LLM决策引擎**: 调用大语言模型生成决策
- **规则校验引擎**: 验证行为符合道义现实主义规则
- **环境仿真引擎**: 管理仿真环境和事件
- **数据存储引擎**: 持久化存储仿真数据
- **日志记录引擎**: 记录所有操作和决策理由

### 2. 核心要素层

定义仿真中的核心实体：

- **实力计算系统**: 基于克莱因方程计算国力
- **智能系统**: 大国、小国、国际组织智能体
- **互动规则**: 国家间互动行为规则
- **记忆系统**: 带访问控制的记忆管理

### 3. 观测迭代层

负责数据分析和流程控制：

- **指标计算**: 自变量、中介变量、因变量计算
- **工作流编排**: 单轮、多轮、并行仿真流程
- **实验管理**: 对照实验和对比分析

### 4. 交互可视化层

提供用户交互界面：

- **FastAPI后端**: RESTful API和WebSocket服务
- **React前端**: 现代化Web应用界面

## 数据流

1. 用户通过前端配置仿真参数
2. 前端调用API启动仿真
3. 后端创建仿真实例
4. 每轮迭代中：
   - LLM生成决策
   - 规则校验验证
   - 执行互动行为
   - 计算指标
   - 保存数据
5. WebSocket推送实时状态
6. 前端更新UI显示

## 技术栈

### 后端
- Python 3.9+
- FastAPI (Web框架)
- SQLAlchemy (ORM)
- SQLite (数据库)
- Pydantic (数据验证)

### 前端
- React 18
- TypeScript 5
- Redux Toolkit (状态管理)
- Vite 5 (构建工具)
- Tailwind CSS 3 (样式)
- Plotly.js (可视化)

### 部署
- Docker
- Docker Compose
- Nginx (反向代理)
