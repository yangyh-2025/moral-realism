# 仿真实验报告：二战前欧洲场景模型校验（ID2）

## 一、实验目的

本实验为模型校验后测，核心目标是检验仿真系统在"糊名"条件下（智能体仅知悉彼此的实力属性、战略关系与地理邻接状态，不获知真实历史身份），生成的国际行为分布与秩序演化轨迹是否与1938年二战前欧洲的历史态势具有结构性一致。相较于ID1（一战前场景），本实验选取的1938年场景具有更为紧迫的危机氛围——苏德对立、轴心国扩张、英法绥靖等历史要素为模型校验提供了更丰富的对照维度。具体校验维度包括：

1. **阵营分化与追随格局**：仿真是否复现了历史中对立阵营的形成过程，以及核心国家聚集盟友/追随者的能力差异。特别关注1938年苏德对立、英法同盟与轴心国体系的阵营结构。

2. **重点国家行为谱系**：仿真中CINC排名前列的国家是否展现出与历史中大国（苏联、德国、英国等）相似的行为模式（军事威慑、外交恫吓、绥靖政策、轴心国合作等）。

3. **全局行为分布**：全体系的行为类别构成是否与二战前欧洲"高对抗、低合作"的危机氛围一致。

4. **冲突升级路径**：主权尊重率下降轨迹与关系恶化路径是否与从外交危机走向战争的历史进程对应。

### 2.2 初始战略关系

初始战略关系矩阵依据1938年历史阵营结构设定。与ID1（一战前）相比，1938年场景的国际体系已呈现出更为清晰的意识形态对立和地缘集团分化：苏联与德国在意识形态上尖锐对立；英国和法国维持传统协约国关系；意大利与德国形成轴心国合作；波兰处于苏德之间的缓冲地带地位；东欧小国面临选边压力。

**关键发现**：

- **苏联（强国甲，强权型）**：初始CINC=1.000 → 最终CINC=0.111，下降**88.9%**。作为初始实力最强的国家，其国力在仿真中被大幅削弱，反映了强权型国家偏好军事对抗导致国力消耗的模式。
- **德国（强国乙，霸权型）**：初始CINC=0.450 → 最终CINC=0.107，下降**76.2%**。德国的国力衰退与历史中二战后期德国资源枯竭的轨迹在方向上对应。

- **多个小国CINC大幅上升**：如挪威+2212.3%、芬兰+2206.3%、爱尔兰+1508.2%（比例结构导致的被动膨胀，因大国国力下降后小国的相对份额增加）。

#### 与历史对比：追随格局

**一致性表现**：

1. **英国主导阵营的形成**：仿真中英国在第2轮即获得21国追随，并在多数轮次维持大量追随者。这与历史中1938年英国在欧洲外交中的核心地位对应——英国通过绥靖政策、外交协调和同盟体系维系着对欧洲小国的影响力。仿真中英国对法国提供援助28次、对波兰17次，复现了历史中英国对盟友的经济支持。

2. **阵营分化的动态波动**：与ID1中追随格局"第4轮即固化"不同，ID2的追随格局呈现出显著的动态波动——第10轮19国中立、第20轮法国意外获得7个追随者、第36-37轮德国聚集11-12个追随者。这种波动与历史中二战前阵营的不稳定性对应：1938年慕尼黑协定后欧洲格局剧烈变动，1939年苏德互不侵犯条约更彻底重塑了阵营结构。

3. **德国阶段性的追随聚集**：第36轮（11国追随德国）、第37轮（12国）、第40轮（9国）显示德国在特定阶段具备吸引追随者的能力。这与历史中1938-1939年轴心国体系扩张（德国吞并奥地利、捷克斯洛伐克后影响力上升）在时序特征上对应。

**不一致性表现**：

1. **最终格局过于一边倒**：第50轮21国追随英国、0国追随德国/苏联，形成"一极独大"而非历史中1938年的"多极对峙"。历史中1938年欧洲存在多个对立集团——英法同盟、轴心国（德意日）、苏联阵营、美国孤立主义——仿真未能复现这种多极竞争格局。

2. **苏联始终未能聚集追随者**：仿真中苏联最高仅获得7国追随（第3、19轮），且最终为0国。历史中苏联虽然在1938年因意识形态孤立而盟友较少，但1939年《苏德互不侵犯条约》后迅速获得势力范围（波罗的海三国、波兰东部）。仿真缺乏此类重大外交转折。

3. **意大利影响力被低估**：仿真中意大利从未获得任何追随者。历史中意大利作为轴心国核心成员，在巴尔干地区和地中海具有显著影响力（阿尔巴尼亚被占领、对南斯拉夫施压）。

#### 与历史对比：重点国家行为谱系

**一致性表现**：

1. **苏德核心对抗的军事化特征**：苏联对德国的行为中，军事手段（胁迫+交战+展示姿态）合计250次（占对德互动的88.9%），与历史中苏德在意识形态和地缘政治上的尖锐对立对应。1938年苏德虽尚未开战，但双方在西班牙内战（1936-1939）中已分别支持对立阵营，形成"代理战争"态势。仿真中苏德互动的零合作特征与历史中苏德关系的对抗本质一致。

2. **德国的"威胁+援助"双重策略**：德国对外行为中威胁占35.1%，但同时提供援助45次（其中80%给予意大利），精准复现了历史中轴心国体系的"核心-卫星"合作模式——德国通过经济军事援助维系与意大利的同盟关系，同时以威胁性外交扩张势力范围。

3. **英国的"援助主导+口头威慑"模式**：英国行为中提供援助74次（16.2%）、外交合作56次（12.3%），同时威胁147次（32.2%）。这一"软硬兼施"模式与历史中英国的绥靖政策+同盟外交双重策略对应——一方面通过援助维系同盟体系，另一方面以口头威慑试图遏制轴心国扩张。

4. **英国对法、波的重点援助**：英国对法国援助28次、对波兰17次，这与历史中1938-1939年英国对欧洲大陆盟友的安全承诺高度一致。1939年3月英国正式对波兰提供安全保证，仿真中英国对波兰的持续援助预演了这一定向合作模式。

**不一致性表现**：

1. **英国对德国缺乏实质性合作**：仿真中英国对德国仅有1次外交合作，而历史中1938年英国对德国实施绥靖政策（慕尼黑协定允许德国吞并苏台德），存在大量外交接触。仿真未能复现绥靖政策这一关键历史行为模式。

2. **苏联对英国同样缺乏合作**：苏联对英国的22次互动全部为军事对抗类行为。历史中1935年苏联与法国和捷克斯洛伐克签订了互助条约，虽然与英国无直接同盟，但1939年英法苏三方谈判中苏联曾与西方接触。仿真中苏联对西方的零合作过于绝对。

3. **交战行为出现较早**：第8轮即出现"交战/使用常规军事武力"行为，第38-42轮交战行为达到高峰（7-15次/轮）。历史中1938年大国间未爆发直接军事冲突（二战爆发于1939年9月）。仿真中交战行为的常态化表明军事冲突阈值可能过低。

**一致性表现**：

1. **主权尊重率的持续下降趋势**：从轮次1的77.0%降至轮次45的12.4%，整体下降64.6个百分点。与历史中1938-1939年国际秩序持续崩解的轨迹对应——从慕尼黑协定的短暂缓和到1939年3月德国占领捷克斯洛伐克后秩序的彻底崩塌。

2. **"大棒威慑型"秩序的主导地位**：50轮中34轮（68%）处于"大棒威慑型"秩序，与历史中1938年欧洲被强权政治主导的现实一致。慕尼黑协定本身就是"大棒外交"的典型产物——希特勒以战争威胁迫使英法让步。

3. **苏德对抗的持续性与渐进升级**：苏德双边对抗贯穿50轮，从初期的展示军事姿态逐步升级为胁迫/强制，后期出现交战行为。这与历史中苏德关系从意识形态对立（1933年希特勒上台后）到地缘政治对抗（西班牙内战 proxy war）再到最终战争（1941年巴巴罗萨计划）的渐进升级路径在结构上对应。

**不一致性表现**：

1. **缺乏"缓和—再危机"的周期性波动**：历史中1938年慕尼黑协定后出现了短暂的"慕尼黑精神"缓和期（1938年10月-1939年3月），随后因德国占领捷克斯洛伐克而急剧恶化。仿真中主权尊重率虽有小幅波动（轮次26升至31.0%），但缺乏这种与重大外交事件对应的周期性波动。

2. **交战行为出现过早且过于频繁**：第8轮即出现交战行为，第38-42轮达到高峰。历史中1938年大国间未发生直接军事冲突，德国的扩张主要通过"不流血的政变"（德奥合并、苏台德地区）实现。仿真中交战行为的常态化混淆了"军事威慑"与"实际战争"的界限。

3. **法国意外获得追随者的不合理性**：第20轮法国获得7国追随者。历史中1938年法国虽为欧洲大国，但因国内政治动荡和军事战略保守（马奇诺防线依赖），其领导力远逊于英国。仿真中法国短暂成为领导者的现象缺乏历史依据。

| 英国主导地位 | **高** | 英国成为核心领导者并获得大量追随，与历史中英国在欧洲外交中的枢纽地位对应 |
| 苏德核心对抗 | **高** | 苏德双边对抗贯穿全程，军事手段占绝对主导，复现了历史中两大意识形态阵营的对立逻辑 |
| 德国的轴心国合作 | **高** | 德国对意大利提供援助36次（占其全部援助的80%），精准复现了轴心国体系的合作模式 |

### 4.2 通过/不通过判定

综合上述十一个维度的评估，本次模型校验的结论如下：

**通过校验的维度**（7项）：阵营分化动态波动、英国主导地位、苏德核心对抗、德国轴心国合作、威胁外交主导、主权尊重率下降、英国对盟友定向援助。这些维度的通过表明仿真系统在**宏观行为谱系**和**秩序演化方向**上具备了与二战前历史对应的基本能力。

**未通过校验的维度**（4项）：多极格局复现、绥靖政策缺失、交战行为频率、冲突升级时间尺度。这些偏差的共同指向是：仿真中的**阵营吸引力机制**存在偏差（德国/苏联难以持续聚集追随者），且**外交行为谱系**不够丰富（缺乏绥靖、外交斡旋等"灰色地带"策略），导致体系未能复现历史中1938年多极对峙的复杂格局。

**总体判定**：**条件通过**。仿真系统在描述性层面（行为分布、秩序类型、大国对抗模式）与二战前历史具有合理对应关系，且在**阵营动态波动**和**轴心国合作模式**两个维度上显著优于ID1。但在规范性层面（多极格局复现、绥靖政策模拟、冲突阈值设定）仍存在偏差，需要在后续实验中对以下参数进行校准优化：

1. **领导者吸引力机制**：增强霸权型和强权型国家对特定类型国家的吸引力（如德国对小国的胁迫性吸引力、苏联对意识形态相近国家的吸引力）；
2. **外交行为丰富度**：增加"绥靖""妥协""外交斡旋"等行为类别，使王道型国家的外交策略更加多元；
3. **冲突阈值调整**：提高"交战"行为的出现门槛，使其仅在关系恶化至"战争关系"多轮后才可能触发。

1938年场景在经过参数调整后，可作为后续理论假说检验实验的效度验证平台。
