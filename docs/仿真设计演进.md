# 仿真设计的演进与方法论选择

> 本节回顾道德现实主义 ABM 系统在原型迭代中遇到的关键设计抉择。对每一个抉择，我们说明早期采用的方案、该方案暴露的方法论缺陷、最终采纳的替代方案，以及在科学合理性、可解释性与可证伪性上前者不如后者的具体理由。这些说明并非工程实现层面的回顾，而是对仿真本身科学协议的论证。

## 一、研究目标与设计原则

本系统的研究目标是在 LLM 驱动的多智能体框架下复现"实力分布 + 领导价值取向 ⇒ 行为谱系 ⇒ 国际秩序"这一国际政治动力学过程。它要回答的核心问题并非"如何让仿真跑起来"，而是"实力与道义这两个维度在多大程度上各自独立地塑造国家行为"。该提问决定了仿真设计必须满足三项规范：第一，行为环境必须以真实历史数据为先验锚点，避免参数主观化使得结论被预设；第二，智能体的决策上下文必须以结构化、可版本化的方式表达，使同一场景的多次重复运行严格可比；第三，仿真程序自身不得替代被研究的主体作合理性判断，否则关键反例会被代码屏蔽。这三项规范贯穿了后文每一处设计选择，也是判断"哪一方案更优"的统一标尺。本节按维度顺序展开，每个维度独立叙述早期方案与最终方案，并在末节回到综合论证。

## 二、国力测度的根基性重构：从克莱因方程到 CINC 比例指数

最早的国力模块采用克莱因综合国力方程，将经济、军事、人口、资源、技术等若干指标按预设权重加权求和，得到一个**绝对分值**，再以此分值的高低决定国家的实力层级。这一方案在原型期具有"立刻可用"的便利，但很快在三个方面遭遇了方法论上的困难。其一，权重为研究者主观设定，不同的文献对同一指标给出的权重相差颇大，仿真结果对权重的微小调整高度敏感，这意味着"哪个国家是大国"实质上取决于研究者的预判，而不是数据本身。其二，绝对分值是单方独立计算的——任何一国国力的变化都不会自动地影响其他国家的相对位置，这使得"安全困境"所必需的"他国崛起即我之相对衰落"的感知失去了内生基础，仿真中的国家彼此之间不会形成真正的结构性紧张。其三，克莱因方程缺乏公认的历史数据校准，研究者不得不为每个场景手工设定初值，这种初值无法接受同行复现与外部检验。

最终方案放弃了克莱因方程，转而采用 Correlates of War 项目 NMC v6.0 数据集所定义的 CINC（Composite Index of National Capability）。CINC 把国力定义为体系内六项底层指标——军费、军队人数、钢铁产量、能源消费、总人口、城市人口——各国占体系全部之比的均值。这个定义在三个方面恰好回应了上述三个困难。首先，CINC 是国际政治量化研究领域公认的标准指标，已经被大量经典文献反复使用，研究者无须为权重选择辩护，结果亦便于跨研究横向比较。其次，CINC 在结构上是一个比例值——所有国家的 CINC 之和恒等于一，任一国的指标变化都被动地、自动地改变了其他国家的相对位置；这一**联动性**本身就是建模安全困境与权力转移的内生机制，而不是事后通过规则强加的。第三，COW NMC 数据集提供了一百八十余年、两百余国的真实历史观测，本研究的三套预设场景（1913 年一战前欧洲、1938 年二战前欧洲、1946 年冷战前欧洲）可以直接采用历史观测值作为客观初值，使得仿真结果是否合理可以与已知历史走向作对照检验。克莱因方程对应的"参数选择即结论选择"的方法论风险在此被彻底消除。

## 三、国力层级的演进：从静态绝对分值到体系内动态相对排名

与克莱因绝对分值相耦合的，是早期版本中以"分值大于阈值"硬编码判定大国、中等强国的做法。这种判定在仿真过程中是静态的：即便某国的国力在一系列轮次内大幅增长或衰退，它的层级标签也不会自然变化；研究者要么在每一轮手动调整阈值，要么干脆冻结所有国家的层级。无论哪种处置，都与"大国崛起"和"霸权衰落"这两个本研究最关心的现象相悖——前者要求层级是观测变量，后者要求层级跃迁内生于仿真本身。

最终方案把层级判定改为按 CINC 在体系内的百分位排名：占据前 10% 的国家被识别为超级大国，10%–30% 为大国，30%–60% 为中等强国，其余为小国，且这一判定在每一轮国力重算之后立刻重做。这一改动在概念上直接把"大国是相对而非绝对概念"这一国际关系学常识嵌入了仿真本体。研究者不再需要为任何阈值辩护，层级变化也不再需要任何外部触发——它是 CINC 体系内份额变动的逻辑结果。更重要的是，"霸权衰落"这一被反复讨论但难以在传统 ABM 中清晰量化的宏观叙事，在新方案下获得了可观测、可重复、可证伪的外延：研究者可以直接观察某一国在多少轮内由超级大国跌入大国行列，或者在多场景比较中考察"霸权稳定"是否随领导类型分布而变化。这种内生化是克莱因绝对分值方案所无法企及的。

## 四、决策提示词架构的演进：从单一模板到分层信息池

LLM 驱动的智能体对上下文结构高度敏感，因此"如何把信息呈现给 LLM"本身就是研究方法的一部分。最早的版本采用一个单一文本模板：角色设定、规则约束、行为列表、历史数据、当前态势全部混排在一段长 prompt 中，由 LLM 一次性消化。这一架构很快暴露出三个方面的问题。第一，规则与数据混杂使 LLM 难以分辨什么是必须遵守的硬约束，什么只是供其参考的情境信息，结果是规则被偶尔违反、情境又被偶尔当作硬约束误用。第二，单一模板把全部历史轮次的行为记录一次性灌入，上下文窗口被早期低相关数据迅速占据，关键的近期态势反而被淹没。第三，决策、追随、目标评估、关系演变这四类智能体被迫共用同一份模板，无法体现各自的角色差异，造成认知任务的相互干扰。

中期的调整是把提示词拆分为 system 与 user 两段：前者承担"你是谁、必须遵守哪些规则、按什么格式输出"，后者承担"本轮的数据"。这一拆分带来了一个直接的好处——规则与数据各自独立版本化，使得"规则不变、数据变化"的多场景实验成为可重复的协议。但拆分本身并未解决上下文淹没问题，因此最终方案在 user prompt 中进一步引入了**分层信息池**：在历史数据之前，先放置一组结构化的前置信息块，依次是【当前态势摘要】【实力矩阵】【冲突升级轨迹】【你的战略关系总览】【候选人评估】，然后才是历史行为；历史行为本身亦采用分层聚合策略——较早的轮次只保留关系对级别的极简统计，最近十轮做合作与对抗的次数计数，最近五条则保留含具体行为内容（action_content）的完整记录。

这种最终架构在认知意义上更接近真实外交决策者的工作方式——后者也是优先读取近期态势、明确的盟友与对手清单、清晰的实力对比，而不是回溯几十年的全部互动史。在统计意义上，分层信息池显著降低了 LLM 在不同轮次之间的决策方差：当 system 几乎不变、user 中前置信息块的结构稳定、唯有数据更新时，决策行为的可重复性获得了来自提示词结构层面的保障。这是单一模板方案在原理上就难以达到的，因为其混排结构本身就是噪声源。

## 五、战略关系系统的演进：从缺失到五级关系、双向演变与候选评估

仿真的初版完全没有显式的双边关系字段——国家之间是友是敌只能由 LLM 在每次决策时从历史行为中隐式推断。这个看似"让 LLM 自己理解"的设计实际上同时违反了三项研究规范。其一，路径依赖被破坏：同一对国家的关系在不同轮次被 LLM 推断为不同状态的情况屡有发生，仿真的整体演化轨迹因此变得不稳定，难以重复。其二，研究者失去了对初始联盟格局的操纵权——而"1913 年三国同盟与三国协约的对峙格局"这一类历史快照恰恰必须以显式关系作为初始条件，否则"一战前欧洲场景"与"普通多国仿真"之间没有任何区分意义。其三，安全困境与联盟链式卷入这两个本研究的核心机制，其形式化定义都依赖于"关系是一个有记忆的状态变量"，而非一个由当前决策即兴推断出的标签。

最终方案分三步建立起一个完整的战略关系子系统。首先，把战略关系定义为**显式状态字段**，分为战争、冲突、无外交、伙伴、盟友五个等级，由数据库持久化；其次，加入由 LLM 驱动的**双向关系演变**机制，每轮结束后根据该轮的行为数据对每一对国家评估升降级，并施加两项硬约束——单轮关系最多升降一级以防止跳变，关系变化后需经历稳定期方可再次变动以防止回弹；再次，在追随决策阶段引入**候选评估前置块**，为每个 voter 预先生成"候选人关系警示与双向互动统计"，让 LLM 在选择追随对象之前先核对每个候选人与自己的战略关系。三步合在一起，使关系成为一个被行为塑造、又约束未来行为的闭环状态变量，并通过显式提示降低 LLM 在小样本下"读漏"隐含关系的概率。

为何最终方案更优可以从可证伪性角度直接回答。隐式推断方案下，关系本身是 LLM 的输出，而 LLM 的输出又依赖于关系——这种循环使得"联盟链式卷入"无论是否在仿真中发生，都难以归因到具体的机制。显式状态变量加上单轮升降一级的硬约束，把"关系演变"从一个不可见的推断过程变成了一个可观察、可统计、可被外部对照（如对比历史上的同盟形成节点）的过程；候选评估前置块进一步把"投票时是否考虑了既有战略关系"这一研究子问题从模型黑箱中分离出来，使后续的反事实分析（如"若关闭关系前置块，追随分布如何变化"）成为可能。

## 六、地理与邻居约束的引入：从纯实力对比到力量投送限制

在引入邻居关系之前，仿真完全不考虑空间因素：任意国家都可以对任意国家施加任意行为，唯一的成本来自行为类型本身与施动方的实力。这种"扁平化"的设定在数学上简洁，但在政治意义上失真。1913 年的塞尔维亚与日本之间几乎不存在直接军事干预的可能性，把这两国置于同一互动平面上会产生大量在历史上不可能发生的事件，仿真分布因此与任何已知历史走向都难以建立映射关系。更重要的是，权力转移与同盟形成这两个被反复强调的核心机制，在国际关系理论中本就强烈依赖于地理邻接：邻国之间的安全困境远比远国之间剧烈，而同盟在很大程度上是地理压力的回应。缺失这一维度的仿真，事实上把现实主义国关理论的一半基础掏空了。

最终方案以最小代价补回了这一维度：为国家间引入二元邻接判定（陆邻接与海邻接），并在军事手段类行为的成本函数中对非邻国附加一项距离惩罚；三套历史预设场景在入库时一并入库了相应年代的陆海邻接矩阵以及与 COW 国家代码（ccode）的映射。这一最小化设计有意识地避免了引入精确距离、地形修正等额外参数，原因是后者会带来与 CINC 框架不一致的参数膨胀，使得"力量投送约束的影响"无法从其他参数干扰中干净地分离出来。引入邻居约束后，仿真生成的行为分布显著向历史可能性收敛——非邻国之间的直接军事行为大幅减少，而邻国对的紧张演化轨迹与历史记录的对照变得有意义。这一改动以"使用最少参数表达最多结构"的原则保护了模型的可解释性。

## 七、决策合规校验的简化：从三层校验到极简校验

最早的决策后处理逻辑包含三层校验：Tier 1 检查 JSON 结构合法性，Tier 2 检查业务字段是否在白名单内并满足类型约束，Tier 3 则进一步检查"规则合规性"，例如"凡是损害国力的行为必须在成本收益分析中显式说明"。Tier 3 看似是一道质量保障，但在方法论上实质上是用程序化规则去**预先判断决策的合理性**。这一逻辑很快遭到了反例的反弹：本研究最关心的现象之一，是不同领导类型（王道、霸权、强权、昏庸）的国家是否会做出不同的、有时甚至与短期国力计算相悖的选择。一个昏庸型领导可能因情绪做出明显不合算的对抗行为，一个王道型领导可能为价值原则承受短期国力代价。Tier 3 把这些反例统统认定为"违规"而丢弃，结果是仿真中保留下来的全部决策都是"合乎短期功利计算"的——而这恰恰把本研究要观测的因变量在数据生成阶段就消灭掉了。

最终方案删除了 Tier 3，只保留 Tier 1 与 Tier 2。这一改动在表面上让校验变弱，在方法论上却让仿真变强：它把"什么样的决策算合理"的判断权交还给 LLM 与领导类型的先验，而不是由仿真程序代为决定。如果"价值取向不影响行为"这一零假设为真，那么反例本应在统计层面被淘汰；如果它不为真，则反例本身就是研究证据。让程序代替统计去屏蔽反例，等同于在数据生成阶段就否定了被检验的假说——这与道德现实主义假说的可证伪性诉求直接冲突。Tier 3 的移除因此不是为了简化代码，而是为了让证据空间向假说真实开放。

## 八、仿真控制与场景管理的演进：从可变初值到历史快照保护

早期版本的仿真重置（reset_simulation）逻辑遵循一种朴素的"清零"理解——把项目相关的所有运行时数据，包括用户配置或场景预设的初始战略关系，一并清空。这一处理在工程上简洁，在科学协议上却破坏了重要的可重复性。一个研究者在一个场景下跑出某条轨迹，之后调用 reset 重置，再用看似相同的配置重新启动仿真，得到的初始关系矩阵已经不再与第一次相同——因为初始关系在第一次 reset 时被清空，第二次启动时被默认值（如普遍"无外交"）填回。结果是同一场景的两次运行从初始条件起就发生分叉，但研究者并不一定意识到这一点。与此同时，硬编码的预设场景使得多年代历史快照难以并存，每增加一个场景都需要修改代码。

最终方案在两个层面做了重新设计。第一，重新划清"运行数据"与"项目配置"之间的边界：reset 仅清除运行数据（历史行为、轮次、追随关系、CINC 历史与当前 CINC），而战略关系作为初始条件的一部分被严格保留；如有研究者出于专门目的需要重置关系，需调用一个独立的、显式的接口。这一设计把"初始条件是科学实验的协议"这一规范落到了系统行为层面。第二，预设场景从硬编码改为持久化在数据库中的三套基于真实 CINC 与邻接数据的历史快照——1913 年一战前欧洲 19 国、1938 年二战前欧洲 28 国、1946 年冷战前欧洲 25 国。这些场景不仅提供了客观初值，也使得仿真结果可以与已知历史事件（如萨拉热窝危机引发的链式同盟卷入、1938 年绥靖政策下的实力转移、1946 年冷战起源的两极分化）作系统化的对照分析。可重复性与可对照性在此都从一个理念变成了系统协议。

## 九、决策执行与战略目标评估的引入

仿真的决策执行最初是串行的：在每一轮的主动与响应阶段，系统逐个国家地调用 LLM 生成决策，等待返回后再处理下一个。对一个由近二十个国家组成的体系而言，单轮的 LLM 调用次数接近四十次，串行执行下一轮就需要约十分钟以上，50 轮即近 9 小时。这本身似乎是一个工程性能问题，但它实际上对实验方法论造成了真实约束：当一个完整批量实验（多场景 × 多领导类型组合 × 多重复运行）需要数周时间，研究者就被迫减少重复次数，导致统计功效不足，置信区间被迫放宽甚至省略。最终方案改为基于信号量的并发执行（默认并发度为 5，可在 1–20 之间配置）加上单个 agent 异常隔离——单点 LLM 调用失败不再阻断整体仿真，仅记录在日志中。这一调整把单轮决策从约 11 分钟压缩到约 2 分钟，使得 30 次以上的重复运行求置信区间在小时尺度上实际可行。这意味着"统计上可信的结论"从原本的不可达变为可达。

与此并列的另一项重要演进，是战略目标评估机制的引入。早期版本没有为"国家利益是否达成"提供任何量化指标，研究结果只能定性叙述——某国在仿真末期"扩展了影响力"或"陷入了孤立"，但缺少可比较的尺度。最终方案采用**双维度加权**：一半的评估权重来自经 Min-Max 标准化的国力贡献（一个相对客观、与 CINC 直接挂钩的量化指标），另一半权重来自 LLM 对该国行为有效性的判断（一个允许吸纳价值取向的主观维度）。为了抑制 LLM 评分的主观漂移，提示词中加入了三档 Few-shot 锚点——85 分、55 分、25 分的具体示例——把无监督回归约束到一个可比较的相对刻度上。纯客观评估会忽视王道型与霸权型在同一国力增长下含义的差异，纯主观评估又会受 LLM 自身漂移影响而难以横向比较；双维度加权与 Few-shot 锚点的结合在两端之间找到了一个对研究问题敏感且对评估方差可控的折中。

## 十、综合论证与方法论意义

上述九项演进若分开看似乎都是局部的修订，但放在一起则呈现出一条清晰的主线：让仿真行为既受真实历史约束、又对价值取向敏感、且具备可证伪的观测窗口。CINC 比例指数与体系内动态层级保证了实力维度有真实历史先验且具备结构性联动；分层信息池与战略关系状态变量保证了智能体的决策上下文是可版本化、可重复的；地理与邻居约束把现实主义国关理论的空间维度补回了仿真本体；移除 Tier 3 合规校验则把可证伪性从代码逻辑中解放出来，让反例有机会成为证据而不是噪声；reset 行为的重新划分与预设场景的数据库化把"初始条件即实验协议"这一规范落地；并发执行让统计上的可信结论实际可达；战略目标评估的双维度与 Few-shot 锚点让"国家利益是否达成"成为一个跨场景可比的量。

更广义地看，本系统最终选择的设计避免了两类常见的极端：一类是纯规则驱动的 ABM——其中行为模式由参数预设、结论由参数选择决定，因而"实力与道义谁更重要"的问题在仿真启动前就已经被答案预设；另一类是纯 LLM 驱动的仿真——其中行为完全由模型即兴产生、状态变量隐藏在 prompt 中，因而结果难以重复、难以解释、难以证伪。本系统介于二者之间——它以真实历史数据作为外生先验，以显式状态变量作为内生记忆，以结构化提示词约束 LLM 决策的认知结构，并以最小化的程序规则保护证据空间向假说真实开放。这种混合架构既不替智能体作合理性判断，也不让仿真陷入完全的随机漂移；它把"实力分布与价值取向是否构成国家行为的两个独立维度"这一可证伪命题，转化成了一个具体的、可执行的、可重复的实验协议。这是本研究希望为后续国际关系 ABM 研究提供的一个方法论范式。
