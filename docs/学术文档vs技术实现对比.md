# 修改计划：对齐学术文档，移除额外限制

## 问题

1. 代码中定义了学术文档中不存在的限制字段：
   - `allowed_initiator_level` - 允许发起该行为的实力层级
   - `forbidden_leader_type` - 禁止执行该行为的领导类型

2. 行为描述被简化，与学术文档不一致

## 学术文档中的完整行为集
